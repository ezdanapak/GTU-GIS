## რა არის QGIS სერტიფიკატი?

**QGIS Certification Programme** არის ოფიციალური პროგრამა, რომელიც QGIS-ის საერთაშორისო ორგანიზაციამ შექმნა ხარისხიანი GIS განათლების წახალისებისა და QGIS საზოგადოების გაძლიერებისთვის.

სერტიფიკატი გაძლევს **საერთაშორისოდ აღიარებულ დადასტურებას** იმისა, რომ გაქვს QGIS-ის პრაქტიკული ცოდნა. სერტიფიკატი:

- ✅ **ვერიფიცირებადია** — დამსაქმებელს შეუძლია ონლაინ შეამოწმოს ავთენტურობა
- ✅ **LinkedIn-ზე დასამატებელია** — პროფილი გამოირჩევა
- ✅ **ESRI-ს სერტიფიკატის ღირსეული ალტერნატივაა** — ბევრად ხელმისაწვდომი ფასად
- ✅ **ხელს უწყობს QGIS-ის განვითარებას** — სერტიფიკატის ფასის ნაწილი პროექტს ეთმობა

!!! tip "ლექტორის გამოცდილება"
    ამ გზა გავლილი და სერტიფიცირებული ვარ პირადად — Introduction to QGIS, Advanced QGIS და PyQGIS კურსები. ქვემოთ ჩემი გამოცდილება და რჩევები გიზიარებ. დეტალური სტატია Medium-ზეც შეგიძლია იხილო: [სერტიფიცირება — Giorgi Kapanadze](https://kapanadze.medium.com/%E1%83%A1%E1%83%94%E1%83%A0%E1%83%A2%E1%83%98%E1%83%A4%E1%83%98%E1%83%AA%E1%83%98%E1%83%A0%E1%83%94%E1%83%91%E1%83%90-86e411a7692d)

| 🌍 ინგლისური ენა | კურსები ინგლისურადაა — B1–B2 დონე საკმარისია |

### 📚 კურსების სასწავლო გზა

=== "1️⃣ Introduction to QGIS"

    **დონე:** დამწყები  
    **ვისთვის:** ნულიდან იწყებ, ან გჭირდება სტრუქტურირებული ბაზა  

    **ისწავლი:**
    - QGIS-ის ინტერფეისი და პარამეტრები
    - ვექტორული და რასტრული მონაცემების დამუშავება
    - კოორდინატთა სისტემები და პროექციები
    - ატრიბუტული ცხრილები და ფილტრაცია
    - რუკის კომპოზიცია და ექსპორტი
    - გეოდამუშავების ძირითადი ოპერაციები

    ✅ **შედეგი:** QGIS-ის საბაზისო სერტიფიკატი

=== "2️⃣ Advanced QGIS"

    **დონე:** შუალედური  
    **წინაპირობა:** Introduction to QGIS დასრულება  

    **ისწავლი:**
    - Processing Toolbox-ის სიღრმისეული გამოყენება
    - მოდელების შექმნა (Model Builder)
    - Expression ენა — $geometry, @atlas_feature...
    - Network Analysis — მოკლე გზები, სერვისის ზონები
    - Raster Analysis — DEM, slope, aspect, hillshade
    - PostGIS-თან კავშირი
    - Atlas-ის გამოყენება სერიული რუკებისთვის

    ✅ **შედეგი:** QGIS-ის Advanced სერტიფიკატი

=== "3️⃣ PyQGIS Masterclass"

    **დონე:** მოწინავე  
    **წინაპირობა:** Advanced QGIS + Python-ის ცოდნა  

    **Python-ის წინასწარ გჭირდება:**
    - OOP (კლასები, მეთოდები)
    - ფაილებთან მუშაობა
    - Pandas, NumPy ბიბლიოთეკები

    **ისწავლი:**
    - QGIS-ის Python API (PyQGIS)
    - Plugin-ების შექმნა
    - Processing Script-ები
    - გეოდამუშავების ავტომატიზაცია
    - Custom Tool-ების დაპროგრამება

    ✅ **შედეგი:** PyQGIS სერტიფიკატი

## ❓ ხშირად დასმული კითხვები

??? question "ESRI-ის სერტიფიკატისგან რით განსხვავდება?"
    ESRI-ის (ArcGIS) სერტიფიკაცია ძვირი და კომერციულ პროგრამულ უზრუნველყოფაზეა დაფუძნებული. QGIS-ის სერტიფიკატი ღია კოდის ინსტრუმენტზეა და ბევრად ხელმისაწვდომი — ასევე ის ფაქტი, რომ QGIS ემატება მნიშვნელობით მსოფლიო GIS-ის ბაზარზე.

??? question "რამდენ ხანს გრძელდება სერტიფიკატი?"
    QGIS-ის სერტიფიკატს ვადის გასვლა არ აქვს. ვინაიდან QGIS-ი მუდმივად ვითარდება, ახალი ვერსიების კურსების გავლა და პორტფოლიოს განახლება პრაქტიკული გამოცდილება აქვს.

??? question "ვერ ვიხდი — სტიპენდია შეიძლება?"
    დიახ. Spatial Thoughts გთავაზობს სრულ სტიპენდიას. შეავსე განაცხადი — [Sponsor a Candidate](https://spatialthoughts.com/sponsor/) — ასევე შეგიძლია ლექტორს ჰკითხო, დაგეხმაროს.

??? question "მარტო ქართული ბარათი მაქვს — გადავიხდი?"
    დიახ — Visa და Mastercard ქართული ბარათები მუშაობს.

??? question "შეიძლება მხოლოდ სერტიფიკატი ავიღო კურსის გარეშე?"
    არა. სერტიფიკატი მხოლოდ ავტორიზებული კურსის გავლის შემდეგ გაიცემა. ეს ხარისხს უზრუნველყოფს.

??? question "სერტიფიკატი ქართულ სამუშაო ბაზარზე ღირებულია?"
    GIS-ის სფეროში, გარემოსდაცვით, სივრცით დაგეგმვისა და კადასტრის ორგანიზაციებში — დიახ. სერტიფიკატი გამოირჩევა CV-ში და ამოწმებადია.
