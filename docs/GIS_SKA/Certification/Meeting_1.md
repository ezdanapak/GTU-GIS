## 📚 შეხვედრაზე გავლილი — Module 1: Creating Maps

### სავარჯიშო 1.1 — ვექტორული მონაცემების იმპორტი

**Case study:** მნიშვნელოვანი მიწისძვრები 2000–2020 წლებში

**ისწავლი:**

- `Data Source Manager`-ით ფაილების გახსნა:
    - **Shapefile** (`.shp`) — ვექტორული ფორმატი
    - **GeoPackage** (`.gpkg`) — თანამედროვე ფორმატი
    - **Delimited Text** (`.csv`) — კოორდინატებიანი ცხრილი
- ატრიბუტული ცხრილის (`Attribute Table`) გახსნა და კითხვა
- `Select Features by Value` — ობიექტების შერჩევა მნიშვნელობით
- ფენების პანელი (`Layers Panel`) — ფენების რიგი და ხილვადობა

!!! tip "მთავარი გასაგები"
    **GIS-ში ყველაფერი ფენაა (Layer).** თითოეული ფენა ერთ მონაცემთა წყაროს წარმოადგენს.
    ფენების **რიგი** განსაზღვრავს, რომელი ზემოდან ჩანს.

---

### სავარჯიშო 1.2 — სიმბოლოები (Symbology)

**ისწავლი:**

- `Layer Properties` → `Symbology` — ფენის სტილის შეცვლა
- **Single Symbol** — ყველა ობიექტი ერთნაირი სიმბოლოთი
- **Categorized** — კატეგორიების მიხედვით სხვადასხვა ფერი
- **Graduated** — რიცხვითი მნიშვნელობების მიხედვით ზომა/ფერი
- **პროპორციული წრეები** — Data-driven size (მიწისძვრის სიმძლავრე → წრის ზომა)
- პროექციის შეცვლა **Equal Earth**-ზე (`Project` → `Properties` → CRS)


??? info "რატომ ვცვლით CRS-ს? WGS 84 → Equal Earth (EPSG:8857)"

    მიწისძვრების რუკის შედგენისას საკოორდინატო სისტემის შეცვლა **WGS 84 (EPSG:4326)**-დან
    **WGS 84 / Equal Earth Greenwich (EPSG:8857)**-ზე ძალიან მნიშვნელოვანი ნაბიჯია კარტოგრაფიული თვალსაზრისით.

    ---

    **1. მიზეზი: დამახინჯების თავიდან აცილება**

    **WGS 84 (EPSG:4326)** არ არის პროექცია — ეს არის გეოგრაფიული საკოორდინატო სისტემა.
    როდესაც მას ვიყენებთ რუკაზე, QGIS მას გამოსახავს ისე, თითქოს დედამიწა იყოს **ბრტყელი კვადრატი**.
    ეს იწვევს ფორმების ძლიერ დამახინჯებას, განსაკუთრებით პოლუსებთან ახლოს
    *(მაგ. ანტარქტიდა და გრენლანდია არაბუნებრივად დიდი ჩანს).*

    **Equal Earth** კი არის **ტოლფართობიანი (Equal-area)** პროექცია —
    მისი მთავარი მიზანია, დედამიწის სხვადასხვა ნაწილების **ფართობები რეალური პროპორციით** გამოჩნდეს.

    ---

    **2. ეფექტი: მონაცემთა სწორი აღქმა**

    - **WGS 84-ში:** წერტილების სიმჭიდროვე ვიზუალურად იცვლება განედის მიხედვით —
      ჩრდილოეთის ქვეყნები "გაწელილია", რაც მცდარ შთაბეჭდილებას ქმნის მიწისძვრების გავრცელებაზე.
    - **Equal Earth-ში:** რუკა ინარჩუნებს ფართობების რეალურ თანაფარდობას —
      დამთვალიერებელი სწორად აღიქვამს, სად არის მიწისძვრების რეალური სიმჭიდროვე.

    ---

    **3. ესთეტიკა და პროფესიული სტანდარტი**

    Equal Earth პროექცია შეიქმნა **2018 წელს** და ითვლება ერთ-ერთ ყველაზე თანამედროვე
    და ვიზუალურად სასიამოვნო პროექციად მსოფლიო რუკებისთვის.
    მას აქვს **მომრგვალებული (ოვალური) ფორმა**, რაც დედამიწის სფერულობას უფრო კარგად ასახავს.

    ---

    **QGIS-ში ეს ასე გამოიყურება:**

    პროექტის CRS-ის შეცვლისას *(ქვედა მარჯვენა კუთხეში EPSG კოდზე დაჭერით)*:

    - 🗺️ რუკის ფორმა გახდება **ოვალური/ელიფსური**
    - 🌍 კონტინენტები მიიღებენ **უფრო ბუნებრივ ფორმას**
    - 📍 მიწისძვრის წერტილები **ავტომატურად გადაადგილდება** ახალ კოორდინატებზე

    > **მოკლედ:** ამას აკეთებ იმისათვის, რომ რუკა იყოს **მეცნიერულად უფრო ზუსტი**
    > (ფართობების თვალსაზრისით) და **ვიზუალურად უფრო პროფესიონალური**.

```
Layer Properties → Symbology
    ↓
Graduated → Column: mag (მაგნიტუდა)
    ↓
Method: Size → Min: 1px, Max: 10px
    ↓
Classify → OK
```

!!! note "Data-driven Symbology"
    როდესაც სიმბოლოს **ზომა ან ფერი** მონაცემის მნიშვნელობაზეა დამოკიდებული —
    ეს Data-driven ვიზუალიზაციაა. მიწისძვრების შემთხვევაში: **დიდი მაგნიტუდა = დიდი წრე.**

### სავარჯიშო 1.3 — წარწერები (Labelling)

**ისწავლე:**

- `Layer Properties` → `Labels` → `Single Labels`
- **Expression-ბაზირებული წარწერა** — ატრიბუტების გაერთიანება:
    ```
    "country" || '\n' || "mag"
    ```
    `||` — სიმბოლოების შეერთება (concatenation)  
    `'\n'` — ახალი ხაზი
- წარწერის **ფონი** (background) და **callout line** — ისარი ობიექტიდან ეტიკეტამდე
- **Manual Placement** — წარწერის ხელით გადატანა `Label Toolbar`-ით

!!! tip "QGIS Expression-ები"
    Expression-ები QGIS-ის ყველაზე ძლიერი ხელსაწყოა —
    წარწერებში, სიმბოლოებში, ველების გამოთვლაში. ბევრ სავარჯიშოში გამოჩნდება.

### სავარჯიშო 1.4 — ბეჭდვის განლაგება (Print Layout)

**ისწავლე:**

- `Project` → `New Print Layout` — ახალი განლაგების გახსნა
- ელემენტების დამატება:

    | ელემენტი | ღილაკი / მენიუ |
    |---------|---------------|
    | 🗺️ რუკა | Add Map |
    | 📝 სათაური | Add Label |
    | 🔑 ლეგენდა | Add Legend |
    | 📏 მასშტაბი | Add Scale Bar |
    | 🖼️ სურათი/ლოგო | Add Picture |
    | ⬜ ჩარჩო | Add Shape |

- `Export as Image` — PNG/JPG ფორმატში ექსპორტი
- `Export as PDF` — PDF ფორმატში ექსპორტი
