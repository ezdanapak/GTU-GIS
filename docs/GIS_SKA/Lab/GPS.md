## рЃЊрЃљрЃЋрЃљрЃџрЃћрЃЉрЃўрЃА рЃўрЃюрЃАрЃбрЃарЃБрЃЦрЃфрЃўрЃћрЃЉрЃў

рЃБрЃюрЃўрЃЋрЃћрЃарЃАрЃўрЃбрЃћрЃбрЃўрЃА рЃЎрЃЮрЃЏрЃърЃўрЃБрЃбрЃћрЃарЃћрЃЉрЃќрЃћ рЃўрЃЏрЃБрЃерЃљрЃЋрЃћрЃЌ рЃерЃћрЃЏрЃЊрЃћрЃњ рЃАрЃљрЃЦрЃљрЃдрЃљрЃџрЃЊрЃћрЃерЃў:  
`C:\Users\Public\` рЃљрЃю `C:\Users\Public\Documents`  
*(рЃарЃћрЃЎрЃЮрЃЏрЃћрЃюрЃЊрЃћрЃЉрЃБрЃџрЃўрЃљ рЃърЃўрЃарЃљрЃЊ рЃЎрЃЮрЃЏрЃърЃўрЃБрЃбрЃћрЃарЃћрЃЉрЃќрЃћрЃф, рЃарЃљрЃЊрЃњрЃљрЃю рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃљрЃА рЃАрЃ«рЃЋрЃљ рЃљрЃЊрЃњрЃўрЃџрЃљрЃА рЃљрЃарЃАрЃћрЃЉрЃБрЃџрЃў рЃцрЃљрЃўрЃџрЃћрЃЉрЃўрЃА рЃЊрЃљрЃЏрЃБрЃерЃљрЃЋрЃћрЃЉрЃљ рЃерЃћрЃАрЃљрЃФрЃџрЃЮрЃљ рЃњрЃљрЃБрЃГрЃўрЃарЃЊрЃћрЃА.)*

=== "Step II: Doing task"
* рЃцрЃЮрЃюрЃбрЃћрЃЉрЃў, рЃўрЃюрЃАрЃбрЃљрЃџрЃљрЃфрЃўрЃљ, рЃЏрЃљрЃЌрЃў рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃљ, рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ рЃгрЃљрЃарЃгрЃћрЃарЃћрЃЉрЃќрЃћ.
* GPS рЃЎрЃЮрЃЮрЃарЃЊрЃўрЃюрЃљрЃбрЃћрЃЉрЃў рЃЏрЃљрЃАрЃљрЃџрЃўрЃЊрЃљрЃю рЃЊрЃљрЃљрЃЎрЃЮрЃюрЃЋрЃћрЃарЃбрЃўрЃарЃћрЃЌ рЃерЃћрЃАрЃљрЃЉрЃљрЃЏрЃўрЃА рЃцрЃЮрЃарЃЏрЃљрЃбрЃерЃў, рЃерЃћрЃЏрЃЮрЃўрЃбрЃљрЃюрЃћрЃЌ рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃљрЃерЃў рЃЊрЃљрЃАрЃЋрЃўрЃЌ рЃерЃћрЃАрЃљрЃЉрЃљрЃЏрЃўрЃА рЃАрЃљрЃЎрЃЮрЃЮрЃарЃЊрЃўрЃюрЃљрЃбрЃЮ рЃАрЃўрЃАрЃбрЃћрЃЏрЃљрЃерЃў,
рЃерЃћрЃўрЃюрЃљрЃ«рЃћрЃЌ рЃћрЃЦрЃАрЃърЃЮрЃарЃбрЃќрЃћ рЃЌрЃўрЃЌрЃЮрЃћрЃБрЃџрЃў рЃЏрЃљрЃЌрЃњрЃљрЃюрЃў рЃарЃЮрЃњрЃЮрЃарЃф shapefile, рЃљрЃю рЃњрЃћрЃЮрЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃЉрЃљрЃќрЃўрЃА Feature class. <br>
* рЃерЃћрЃЏрЃЮрЃўрЃбрЃљрЃюрЃћрЃЌ рЃЎрЃЮрЃЮрЃарЃЊрЃўрЃюрЃљрЃбрЃћрЃЉрЃўрЃљрЃюрЃў рЃцрЃЮрЃбрЃЮрЃАрЃБрЃарЃљрЃЌрЃћрЃЉрЃў (Geotaggedphotos) рЃарЃЮрЃњрЃЮрЃарЃф shapefile, рЃљрЃю рЃарЃЮрЃњрЃЮрЃарЃф рЃњрЃћрЃЮрЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃЉрЃљрЃќрЃўрЃА Feature class. <br>
* рЃњрЃљрЃарЃЊрЃљрЃЦрЃЏрЃћрЃюрЃўрЃЌ рЃцрЃЮрЃбрЃЮрЃћрЃЉрЃўрЃА рЃерЃарЃћ PT Assign projection рЃћрЃџрЃўрЃцрЃАрЃЮрЃўрЃЊрЃўрЃЊрЃљрЃю UTM рЃќрЃЮрЃюрЃљрЃерЃў рЃЊрЃљ рЃњрЃљрЃЏрЃЮрЃЌрЃЋрЃљрЃџрЃћрЃЌ XY рЃЎрЃЮрЃЮрЃарЃЊрЃўрЃюрЃљрЃбрЃћрЃЉрЃў Add geometry attributes рЃерЃарЃўрЃА рЃфрЃ«рЃарЃўрЃџрЃерЃў. <br>
* рЃљрЃўрЃдрЃћрЃЌ рЃцрЃЮрЃбрЃЮ рЃЎрЃЮрЃЮрЃарЃЊрЃўрЃюрЃљрЃбрЃћрЃЉрЃўрЃА рЃњрЃљрЃарЃћрЃерЃћ рЃЊрЃљ рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃўрЃЌрЃў рЃ«рЃћрЃџрЃАрЃљрЃгрЃДрЃЮрЃћрЃЉрЃўрЃА рЃЊрЃљрЃ«рЃЏрЃљрЃарЃћрЃЉрЃўрЃЌ рЃЏрЃўрЃљрЃюрЃўрЃГрЃћрЃЌ рЃљрЃЊрЃњрЃўрЃџрЃЏрЃЊрЃћрЃЉрЃљрЃарЃћрЃЮрЃЉрЃљ [geoimgr](https://tool.geoimgr.com/), [GEOSETTER](https://geosetter.de/en/main-en/). <br>
* SVG рЃЏрЃљрЃарЃЎрЃћрЃарЃћрЃЉрЃў, рЃЕрЃљрЃЏрЃЮрЃгрЃћрЃарЃљ рЃЌрЃљрЃЋрЃўрЃАрЃБрЃцрЃљрЃџрЃў рЃгрЃЋрЃЊрЃЮрЃЏрЃўрЃЌ, рЃерЃћрЃюрЃљрЃ«рЃЋрЃљ рЃЊрЃљ рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ рЃгрЃћрЃарЃбрЃўрЃџрЃЮрЃЋрЃљрЃю рЃњрЃћрЃЮрЃЏрЃћрЃбрЃарЃўрЃљрЃќрЃћ рЃ«рЃљрЃбрЃБрЃџрЃљрЃЊ. <br>
* рЃерЃћрЃўрЃюрЃљрЃ«рЃћ рЃерЃарЃўрЃА рЃАрЃбрЃўрЃџрЃў рЃарЃЮрЃњрЃЮрЃарЃф QGIS qml рЃцрЃЮрЃарЃЏрЃљрЃбрЃерЃў рЃљрЃАрЃћрЃЋрЃћ SLD рЃцрЃљрЃўрЃџрЃљрЃЊ. <br>
* рЃерЃћрЃўрЃюрЃљрЃ«рЃћрЃЌ рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃцрЃљрЃўрЃџрЃў рЃАрЃљрЃ«рЃћрЃџрЃўрЃЌ `"Chiatura_GPS_project.qgz"`. рЃљрЃАрЃћрЃЋрЃћ рЃерЃћрЃўрЃюрЃљрЃ«рЃћрЃЌ  `"Chiatura_GPS_project.qgs"` рЃцрЃЮрЃарЃЏрЃљрЃбрЃерЃў. <br>
