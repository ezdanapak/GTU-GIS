## рЃЕрЃЋрЃћрЃю рЃЋрЃўрЃДрЃћрЃюрЃћрЃЉрЃЌ:

- **GeoServer** | рЃЋрЃћрЃарЃАрЃўрЃљ: [GeoServer 2.25.x](https://geoserver.org/download/)

- GeoServer рЃљрЃарЃўрЃА рЃдрЃўрЃљ рЃЎрЃЮрЃЊрЃўрЃА рЃАрЃћрЃарЃЋрЃћрЃарЃБрЃџрЃў рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃљ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃерЃћрЃЦрЃЏрЃюрЃўрЃџрЃўрЃљ рЃњрЃћрЃЮрЃњрЃарЃљрЃцрЃўрЃБрЃџрЃў рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃА рЃЋрЃћрЃЉрЃќрЃћ рЃњрЃљрЃАрЃљрЃЋрЃарЃфрЃћрЃџрЃћрЃЉрЃџрЃљрЃЊ. рЃўрЃА рЃўрЃДрЃћрЃюрЃћрЃЉрЃА **OGC рЃАрЃбрЃљрЃюрЃЊрЃљрЃарЃбрЃћрЃЉрЃА** (WMS, WFS, WCS) рЃЊрЃљ рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃљрЃА рЃўрЃФрЃџрЃћрЃЋрЃљ рЃАрЃўрЃЋрЃарЃфрЃБрЃџрЃў рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃА рЃгрЃљрЃЎрЃўрЃЌрЃ«рЃЋрЃљ, рЃАрЃћрЃарЃЋрЃўрЃарЃћрЃЉрЃљ рЃЊрЃљ рЃАрЃбрЃўрЃџрЃўрЃќрЃљрЃфрЃўрЃљ рЃАрЃ«рЃЋрЃљрЃЊрЃљрЃАрЃ«рЃЋрЃљ рЃцрЃЮрЃарЃЏрЃљрЃбрЃўрЃЊрЃљрЃю, рЃарЃЮрЃњрЃЮрЃарЃўрЃфрЃљрЃљ Shapefile, GeoTIFF, PostGIS рЃЊрЃљ рЃАрЃ«рЃЋрЃљ.

GeoServer рЃљрЃЦрЃбрЃўрЃБрЃарЃљрЃЊ рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ рЃЋрЃћрЃЉ рЃарЃБрЃЎрЃћрЃЉрЃўрЃА рЃърЃџрЃљрЃбрЃцрЃЮрЃарЃЏрЃћрЃЉрЃќрЃћ (рЃЏрЃљрЃњ. Leaflet, OpenLayers), рЃЊрЃљ рЃўрЃюрЃбрЃћрЃњрЃарЃўрЃарЃЊрЃћрЃЉрЃљ рЃўрЃАрЃћрЃЌ рЃбрЃћрЃЦрЃюрЃЮрЃџрЃЮрЃњрЃўрЃћрЃЉрЃЌрЃљрЃю, рЃарЃЮрЃњрЃЮрЃарЃўрЃфрЃљрЃљ QGIS, GeoNode, MapStore, Django рЃЊрЃљ рЃЏрЃарЃљрЃЋрЃљрЃџрЃў рЃАрЃ«рЃЋрЃљ. рЃўрЃњрЃў рЃ«рЃћрЃџрЃЏрЃўрЃАрЃљрЃгрЃЋрЃЊрЃЮрЃЏрЃўрЃљ рЃарЃЮрЃњрЃЮрЃарЃф рЃЊрЃљрЃЏрЃЮрЃБрЃЎрЃўрЃЊрЃћрЃЉрЃћрЃџрЃў рЃАрЃћрЃарЃЋрЃћрЃарЃў, рЃљрЃАрЃћрЃЋрЃћ рЃўрЃюрЃбрЃћрЃњрЃарЃўрЃарЃћрЃЉрЃБрЃџрЃўрЃљ рЃЏрЃарЃљрЃЋрЃљрЃџрЃдрЃўрЃарЃћрЃЉрЃБрЃџ рЃАрЃўрЃАрЃбрЃћрЃЏрЃћрЃЉрЃерЃў.

---

!!!tip  
    GeoServer рЃЏрЃБрЃерЃљрЃЮрЃЉрЃА Java-рЃќрЃћ рЃЊрЃљрЃцрЃБрЃФрЃюрЃћрЃЉрЃБрЃџ рЃњрЃљрЃарЃћрЃЏрЃЮрЃерЃў рЃЊрЃљ рЃерЃћрЃАрЃљрЃФрЃџрЃћрЃЉрЃћрЃџрЃўрЃљ рЃЏрЃўрЃАрЃў рЃЊрЃљрЃДрЃћрЃюрЃћрЃЉрЃљ Windows, Linux рЃЊрЃљ macOS-рЃќрЃћ. рЃарЃћрЃЎрЃЮрЃЏрЃћрЃюрЃЊрЃћрЃЉрЃБрЃџрЃўрЃљ рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃЮрЃЌ Java 11 рЃљрЃю 17.

## рЃўрЃюрЃАрЃбрЃљрЃџрЃљрЃфрЃўрЃўрЃА рЃерЃћрЃЏрЃЊрЃћрЃњ

- **рЃерЃћрЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЌ Java рЃњрЃљрЃарЃћрЃЏрЃЮ:**  
    рЃљрЃБрЃфрЃўрЃџрЃћрЃЉрЃћрЃџрЃўрЃљ рЃАрЃгрЃЮрЃарЃљрЃЊ рЃўрЃДрЃЮрЃА рЃЊрЃљрЃДрЃћрЃюрЃћрЃЉрЃБрЃџрЃў **JAVA_HOME** рЃфрЃЋрЃџрЃљрЃЊрЃў рЃЊрЃљ рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃБрЃџрЃў рЃўрЃДрЃЮрЃА рЃЏрЃ«рЃљрЃарЃЊрЃљрЃГрЃћрЃарЃўрЃџрЃў рЃЋрЃћрЃарЃАрЃўрЃљ (OpenJDK рЃљрЃю Temurin).

- **рЃгрЃЋрЃЊрЃЮрЃЏрЃљ рЃЋрЃћрЃЉ рЃўрЃюрЃбрЃћрЃарЃцрЃћрЃўрЃАрЃќрЃћ:**  
    GeoServer-рЃА рЃљрЃЦрЃЋрЃА рЃўрЃюрЃбрЃћрЃњрЃарЃўрЃарЃћрЃЉрЃБрЃџрЃў рЃЋрЃћрЃЉ рЃўрЃюрЃбрЃћрЃарЃцрЃћрЃўрЃАрЃў, рЃарЃЮрЃЏрЃћрЃџрЃАрЃљрЃф рЃЏрЃўрЃЏрЃљрЃарЃЌрЃљрЃЋрЃЌ рЃЉрЃарЃљрЃБрЃќрЃћрЃарЃўрЃЊрЃљрЃю:  
    `http://localhost:8080/geoserver`

- **рЃЊрЃљрЃљрЃЏрЃљрЃбрЃћрЃЌ рЃАрЃўрЃЋрЃарЃфрЃБрЃџрЃў рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃў:**  
    рЃЊрЃљрЃўрЃгрЃДрЃћрЃЌ `Data Stores` рЃЏрЃћрЃюрЃўрЃБрЃЊрЃљрЃю РђЊ рЃЊрЃљрЃБрЃЎрЃљрЃЋрЃерЃўрЃарЃћрЃЌ рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃА рЃгрЃДрЃљрЃарЃЮ (Shapefile, GeoPackage, PostGIS рЃЊрЃљ рЃАрЃ«рЃЋрЃљ).

# ­ЪДГ GeoServer-рЃўрЃА рЃўрЃАрЃбрЃЮрЃарЃўрЃљ

GeoServer рЃљрЃарЃўрЃА рЃдрЃўрЃљ рЃЎрЃЮрЃЊрЃўрЃА Java-рЃАрЃљрЃцрЃБрЃФрЃЋрЃџрЃўрЃљрЃюрЃў рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃљ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃерЃћрЃўрЃЦрЃЏрЃюрЃљ рЃњрЃћрЃЮрЃАрЃўрЃЋрЃарЃфрЃўрЃЌрЃў рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃА рЃЋрЃћрЃЉрЃАрЃћрЃарЃЋрЃўрЃюрЃњрЃўрЃАрЃЌрЃЋрЃўрЃА OGC рЃАрЃбрЃљрЃюрЃЊрЃљрЃарЃбрЃћрЃЉрЃўрЃА рЃЊрЃљрЃфрЃЋрЃўрЃЌ.

## ­ЪЋ░№ИЈ рЃЊрЃљрЃАрЃљрЃгрЃДрЃўрЃАрЃў РђЊ рЃЎрЃЮрЃЊрЃў рЃАрЃљрЃќрЃЮрЃњрЃљрЃЊрЃЮрЃћрЃЉрЃўрЃАрЃњрЃљрЃю

- **2001 рЃгрЃћрЃџрЃў** РђЊ GeoServer-рЃўрЃА рЃърЃарЃЮрЃћрЃЦрЃбрЃў рЃЊрЃљрЃўрЃгрЃДрЃЮ РђюThe Open Planning ProjectРђЮ-рЃўрЃА рЃўрЃюрЃўрЃфрЃўрЃљрЃбрЃўрЃЋрЃўрЃЌ, рЃЊрЃљ рЃњрЃљрЃ«рЃЊрЃљ рЃћрЃарЃЌ-рЃћрЃарЃЌрЃў рЃърЃўрЃарЃЋрЃћрЃџрЃў OGC WFS рЃАрЃћрЃарЃЋрЃћрЃарЃўрЃА рЃарЃћрЃљрЃџрЃўрЃќрЃљрЃфрЃўрЃљ.
- рЃЏрЃўрЃќрЃљрЃюрЃў рЃўрЃДрЃЮ рЃцрЃљрЃарЃЌрЃЮрЃЊ рЃ«рЃћрЃџрЃЏрЃўрЃАрЃљрЃгрЃЋрЃЊрЃЮрЃЏрЃў рЃЊрЃљ рЃАрЃбрЃљрЃюрЃЊрЃљрЃарЃбрЃћрЃЉрЃЌрЃљрЃю рЃЌрЃљрЃЋрЃАрЃћрЃЉрЃљрЃЊрЃў рЃњрЃћрЃЮрЃАрЃћрЃарЃЋрЃћрЃарЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃљ.

## ­Ъџђ рЃњрЃљрЃюрЃЋрЃўрЃЌрЃљрЃарЃћрЃЉрЃљ рЃЊрЃљ рЃќрЃарЃЊрЃљ

- **2006** РђЊ GeoServer рЃўрЃгрЃДрЃћрЃЉрЃА WMS-рЃўрЃА рЃАрЃарЃБрЃџ рЃЏрЃ«рЃљрЃарЃЊрЃљрЃГрЃћрЃарЃљрЃА рЃЊрЃљ рЃўрЃюрЃбрЃћрЃњрЃарЃўрЃарЃЊрЃћрЃЉрЃљ Google Earth-рЃЌрЃљрЃю.
- **2008** РђЊ рЃўрЃгрЃДрЃћрЃЉрЃљ рЃљрЃЦрЃбрЃўрЃБрЃарЃў рЃўрЃюрЃбрЃћрЃњрЃарЃљрЃфрЃўрЃљ PostGIS-рЃЌрЃљрЃю рЃЊрЃљ рЃАрЃћрЃарЃЋрЃўрЃАрЃћрЃЉрЃўрЃА рЃњрЃљрЃцрЃљрЃарЃЌрЃЮрЃћрЃЉрЃљ.
- **2010** РђЊ GeoServer рЃ«рЃЊрЃћрЃЉрЃљ OSGeo-рЃА рЃЮрЃцрЃўрЃфрЃўрЃљрЃџрЃБрЃарЃў рЃърЃарЃЮрЃћрЃЦрЃбрЃў.
- **2020-рЃЊрЃљрЃю** РђЊ рЃўрЃгрЃДрЃћрЃЉрЃљ WPS (Web Processing Service), INSPIRE, рЃЊрЃљ REST API рЃерЃћрЃАрЃљрЃФрЃџрЃћрЃЉрЃџрЃЮрЃЉрЃћрЃЉрЃўрЃА рЃЊрЃљрЃ«рЃЋрЃћрЃгрЃљ.

## ­ЪїЇ GeoServer рЃЊрЃдрЃћрЃА

- РюЁ рЃЏрЃ«рЃљрЃарЃА рЃБрЃГрЃћрЃарЃА WMS, WFS, WCS, WMTS рЃАрЃбрЃљрЃюрЃЊрЃљрЃарЃбрЃћрЃЉрЃА  
- РюЁ рЃљрЃћрЃарЃЌрЃўрЃљрЃюрЃћрЃЉрЃА рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃА рЃЏрЃарЃљрЃЋрЃљрЃџ рЃгрЃДрЃљрЃарЃЮрЃЊрЃљрЃю: Shapefile, GeoPackage, PostGIS, Oracle Spatial рЃЊрЃљ рЃАрЃ«рЃЋрЃљ  
- РюЁ рЃљрЃЦрЃЋрЃА рЃЏрЃЊрЃўрЃЊрЃљрЃарЃў рЃАрЃбрЃўрЃџрЃўрЃќрЃљрЃфрЃўрЃўрЃА рЃерЃћрЃАрЃљрЃФрЃџрЃћрЃЉрЃџрЃЮрЃЉрЃћрЃЉрЃў SLD рЃЊрЃљ CSS рЃАрЃбрЃўрЃџрЃўрЃЌ  
- РюЁ рЃЏрЃБрЃерЃљрЃЮрЃЉрЃА Docker, Tomcat рЃЊрЃљ Spring Boot рЃњрЃљрЃарЃћрЃЏрЃЮрЃћрЃЉрЃерЃўрЃф  
- РюЁ рЃцрЃљрЃарЃЌрЃЮрЃЊ рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ рЃарЃЮрЃњрЃЮрЃарЃф рЃАрЃљрЃ»рЃљрЃарЃЮ, рЃўрЃАрЃћ рЃЎрЃЋрЃџрЃћрЃЋрЃўрЃЌ рЃЊрЃљ рЃЎрЃЮрЃЏрЃћрЃарЃфрЃўрЃБрЃџ рЃАрЃћрЃЦрЃбрЃЮрЃарЃерЃў

## ­ЪјЊ рЃарЃљрЃбрЃЮрЃЏ GeoServer?

- ­ЪњИ **рЃАрЃарЃБрЃџрЃўрЃљрЃЊ рЃБрЃцрЃљрЃАрЃЮрЃљ рЃЊрЃљ рЃљрЃа рЃљрЃЦрЃЋрЃА рЃџрЃўрЃфрЃћрЃюрЃќрЃўрЃБрЃарЃў рЃерЃћрЃќрЃдрЃБрЃЊрЃЋрЃћрЃЉрЃў**  
- ­Ъїљ рЃЏрЃ«рЃљрЃарЃА рЃБрЃГрЃћрЃарЃА рЃАрЃљрЃћрЃарЃЌрЃљрЃерЃЮрЃарЃўрЃАрЃЮ рЃАрЃбрЃљрЃюрЃЊрЃљрЃарЃбрЃћрЃЉрЃА рЃЊрЃљ рЃўрЃюрЃбрЃћрЃњрЃарЃўрЃарЃЊрЃћрЃЉрЃљ рЃДрЃЋрЃћрЃџрЃљ рЃгрЃљрЃЏрЃДрЃЋрЃљрЃю GIS рЃўрЃюрЃАрЃбрЃарЃБрЃЏрЃћрЃюрЃбрЃЌрЃљрЃю  
- ­ЪД░ рЃЏрЃФрЃџрЃљрЃЋрЃарЃў рЃЊрЃљ рЃЏрЃЮрЃЦрЃюрЃўрЃџрЃў REST API  
- ­ЪЌ║№ИЈ рЃБрЃќрЃарЃБрЃюрЃЋрЃћрЃџрЃДрЃЮрЃцрЃА рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃА рЃАрЃбрЃўрЃџрЃўрЃќрЃљрЃфрЃўрЃљрЃА рЃЊрЃљ рЃърЃарЃћрЃќрЃћрЃюрЃбрЃљрЃфрЃўрЃљрЃА рЃЋрЃћрЃЉрЃарЃБрЃЎрЃћрЃЉрЃўрЃАрЃљрЃЌрЃЋрЃўрЃА  
- ­ЪЊд рЃ«рЃћрЃџрЃЏрЃўрЃАрЃљрЃгрЃЋрЃЊрЃЮрЃЏрЃўрЃљ рЃарЃЮрЃњрЃЮрЃарЃф WAR, Docker, рЃЊрЃљ рЃАрЃбрЃћрЃюрЃЊрЃљрЃџрЃЮрЃю рЃЋрЃћрЃарЃАрЃўрЃћрЃЉрЃерЃў
