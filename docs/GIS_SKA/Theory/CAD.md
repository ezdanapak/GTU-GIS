
# ­ЪЊЂ CAD рЃЊрЃљ GIS: DXF/DWG рЃцрЃљрЃўрЃџрЃћрЃЉрЃў рЃњрЃћрЃЮрЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃБрЃџ рЃАрЃўрЃАрЃбрЃћрЃЏрЃћрЃЉрЃерЃў

## ­Ъћ╣ рЃерЃћрЃАрЃљрЃЋрЃљрЃџрЃў

**CAD (Computer-Aided Design)** рЃљрЃарЃўрЃА рЃАрЃўрЃАрЃбрЃћрЃЏрЃћрЃЉрЃўрЃА рЃћрЃарЃЌрЃЮрЃЉрЃџрЃўрЃЮрЃЉрЃљ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ рЃўрЃюрЃЪрЃўрЃюрЃарЃўрЃюрЃњрЃерЃў, рЃљрЃарЃЦрЃўрЃбрЃћрЃЦрЃбрЃБрЃарЃљрЃерЃў, рЃњрЃћрЃЮрЃЊрЃћрЃќрЃўрЃљрЃерЃў рЃЊрЃљ рЃЏрЃерЃћрЃюрЃћрЃЉрЃџрЃЮрЃЉрЃљрЃерЃў рЃАрЃўрЃЋрЃарЃфрЃўрЃЌрЃў рЃЮрЃЉрЃўрЃћрЃЦрЃбрЃћрЃЉрЃўрЃА рЃќрЃБрЃАрЃбрЃў рЃЏрЃЮрЃЊрЃћрЃџрЃўрЃарЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА. рЃЏрЃўрЃАрЃў рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃў рЃцрЃЮрЃарЃЏрЃљрЃбрЃћрЃЉрЃўрЃљ:

- **DWG (Drawing):** рЃАрЃљрЃЎрЃБрЃЌрЃарЃћрЃЉрЃўрЃЌрЃў рЃцрЃЮрЃарЃЏрЃљрЃбрЃў, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃерЃћрЃўрЃЏрЃБрЃерЃљрЃЋрЃљ AutoDesk-рЃЏрЃљ.
- **DXF (Drawing Exchange Format):** рЃдрЃўрЃљ рЃбрЃћрЃЦрЃАрЃбрЃБрЃарЃў рЃцрЃЮрЃарЃЏрЃљрЃбрЃў, рЃерЃћрЃЦрЃЏрЃюрЃўрЃџрЃў DWG-рЃА рЃљрЃЊрЃЋрЃўрЃџрЃљрЃЊ рЃњрЃљрЃАрЃљрЃќрЃўрЃљрЃарЃћрЃЉрЃџрЃљрЃЊ рЃАрЃ«рЃЋрЃљ рЃАрЃўрЃАрЃбрЃћрЃЏрЃћрЃЉрЃЌрЃљрЃю.

## ­Ъћ╣ CAD рЃЊрЃљ GIS-рЃўрЃА рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃў рЃњрЃљрЃюрЃАрЃ«рЃЋрЃљрЃЋрЃћрЃЉрЃћрЃЉрЃў

| рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃў       | CAD                             | GIS                                  |
|----------------|----------------------------------|---------------------------------------|
| рЃЏрЃўрЃќрЃљрЃюрЃў          | рЃќрЃБрЃАрЃбрЃў рЃбрЃћрЃЦрЃюрЃўрЃЎрЃБрЃарЃў рЃюрЃљрЃ«рЃљрЃќрЃў           | рЃАрЃўрЃЋрЃарЃфрЃўрЃЌрЃў рЃљрЃюрЃљрЃџрЃўрЃќрЃў рЃЊрЃљ рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃЉрЃљрЃќрЃљ    |
| рЃЎрЃЮрЃЮрЃарЃЊрЃўрЃюрЃљрЃбрЃћрЃЉрЃў     | рЃ«рЃерЃўрЃарЃљрЃЊ рЃџрЃЮрЃЎрЃљрЃџрЃБрЃарЃў, рЃњрЃљрЃарЃћрЃерЃћ рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃарЃћрЃЉрЃўрЃАрЃљ | рЃњрЃћрЃЮрЃарЃћрЃцрЃћрЃарЃћрЃюрЃфрЃўрЃарЃћрЃЉрЃБрЃџрЃў, рЃњрЃџрЃЮрЃЉрЃљрЃџрЃБрЃарЃў         |
| рЃцрЃљрЃўрЃџрЃўрЃА рЃцрЃЮрЃарЃЏрЃљрЃбрЃў  | DWG, DXF                         | Shapefile, GeoPackage, GeoJSON, DB   |
| рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃўрЃА рЃбрЃўрЃърЃў| рЃ«рЃљрЃќрЃћрЃЉрЃў, рЃгрЃћрЃарЃбрЃўрЃџрЃћрЃЉрЃў, рЃбрЃћрЃЦрЃАрЃбрЃў         | рЃњрЃћрЃЮрЃЏрЃћрЃбрЃарЃўрЃљ + рЃљрЃбрЃарЃўрЃЉрЃБрЃбрЃБрЃџрЃў рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃў     |

## ­Ъћ╣ DXF/DWG рЃцрЃљрЃўрЃџрЃћрЃЉрЃў GIS-рЃерЃў

GIS рЃАрЃўрЃАрЃбрЃћрЃЏрЃћрЃЉрЃерЃў (рЃЏрЃљрЃњ. QGIS, ArcGIS) рЃ«рЃерЃўрЃарЃљрЃЊ рЃАрЃљрЃГрЃўрЃарЃЮрЃљ **CAD рЃцрЃљрЃўрЃџрЃћрЃЉрЃўрЃА рЃўрЃЏрЃърЃЮрЃарЃбрЃў**, рЃњрЃљрЃюрЃАрЃљрЃЎрЃБрЃЌрЃарЃћрЃЉрЃўрЃЌ рЃЏрЃљрЃерЃўрЃю, рЃарЃЮрЃфрЃљ рЃБрЃюрЃЊрЃљ рЃњрЃљрЃЋрЃљрЃћрЃарЃЌрЃўрЃљрЃюрЃЮрЃЌ рЃњрЃћрЃЮрЃЊрЃћрЃќрЃўрЃБрЃарЃў рЃюрЃљрЃ«рЃљрЃќрЃћрЃЉрЃў рЃАрЃ«рЃЋрЃљ рЃАрЃўрЃЋрЃарЃфрЃўрЃЌ рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃЌрЃљрЃю.

### ­ЪћИ DWG/DXF рЃўрЃЏрЃърЃЮрЃарЃбрЃў QGIS-рЃерЃў

- **QGIS рЃЏрЃ«рЃљрЃарЃЊрЃљрЃГрЃћрЃарЃљ:**
  - QGIS рЃЏрЃћрЃерЃЋрЃћрЃЮрЃЉрЃўрЃЌ рЃерЃћрЃАрЃљрЃФрЃџрЃћрЃЉрЃћрЃџрЃўрЃљ DXF рЃцрЃљрЃўрЃџрЃћрЃЉрЃўрЃА рЃърЃўрЃарЃЊрЃљрЃърЃўрЃа рЃЕрЃљрЃбрЃЋрЃўрЃарЃЌрЃЋрЃљ рЃарЃЮрЃњрЃЮрЃарЃф **Shapefile рЃцрЃћрЃюрЃћрЃЉрЃў** (рЃгрЃћрЃарЃбрЃўрЃџрЃў, рЃ«рЃљрЃќрЃў, рЃърЃЮрЃџрЃўрЃњрЃЮрЃюрЃў).
  - DWG рЃцрЃљрЃўрЃџрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА рЃАрЃљрЃГрЃўрЃарЃЮрЃљ GDAL/OGR-рЃА рЃЏрЃ«рЃљрЃарЃЊрЃљрЃГрЃћрЃарЃљ рЃљрЃю рЃЎрЃЮрЃюрЃЋрЃћрЃарЃбрЃљрЃфрЃўрЃљ DXF-рЃерЃў.

- **рЃърЃарЃЮрЃфрЃћрЃАрЃў:**
  1. Open QGIS Рєњ *Layer* > *Add Layer* > *Add Vector Layer*
  2. рЃљрЃўрЃарЃЕрЃўрЃћ DXF рЃцрЃљрЃўрЃџрЃў
  3. рЃњрЃљрЃЏрЃЮрЃЕрЃюрЃЊрЃћрЃЉрЃљ рЃарЃљрЃЏрЃЊрЃћрЃюрЃўрЃЏрЃћ рЃцрЃћрЃюрЃљ: `entities`, `points`, `lines`, `polygons`, `annotations`.

### ­ЪћИ ArcGIS-рЃерЃў

ArcGIS рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃљрЃА рЃўрЃФрЃџрЃћрЃЋрЃљ CAD рЃцрЃљрЃўрЃџрЃћрЃЉрЃў рЃўрЃЏрЃърЃЮрЃарЃбрЃўрЃарЃЊрЃћрЃА рЃърЃўрЃарЃЊрЃљрЃърЃўрЃа, рЃЌрЃБ рЃўрЃАрЃўрЃюрЃў **рЃАрЃгрЃЮрЃарЃљрЃЊ рЃљрЃарЃўрЃА рЃњрЃћрЃЮрЃарЃћрЃцрЃћрЃарЃћрЃюрЃфрЃўрЃарЃћрЃЉрЃБрЃџрЃў**.

- CAD рЃцрЃљрЃўрЃџрЃў рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃљрЃЊ рЃўрЃ«рЃАрЃюрЃћрЃЉрЃљ CAD Dataset-рЃўрЃА рЃАрЃљрЃ«рЃўрЃЌ.
- рЃерЃћрЃАрЃљрЃФрЃџрЃћрЃЉрЃћрЃџрЃўрЃљ рЃЏрЃўрЃАрЃў рЃЎрЃЮрЃюрЃЋрЃћрЃарЃбрЃљрЃфрЃўрЃљ Feature Class-рЃерЃў.
- рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ *CAD to Geodatabase* рЃљрЃю *Feature Class to Feature Class* рЃ«рЃћрЃџрЃАрЃљрЃгрЃДрЃЮрЃћрЃЉрЃў.

## ­Ъћ╣ рЃњрЃћрЃЮрЃарЃћрЃцрЃћрЃарЃћрЃюрЃфрЃўрЃарЃћрЃЉрЃљ CAD рЃцрЃљрЃўрЃџрЃерЃў

CAD рЃцрЃљрЃўрЃџрЃў рЃљрЃа рЃерЃћрЃўрЃфрЃљрЃЋрЃА рЃДрЃЮрЃЋрЃћрЃџрЃЌрЃЋрЃўрЃА рЃњрЃћрЃЮрЃњрЃарЃљрЃцрЃўрЃБрЃџ рЃАрЃўрЃАрЃбрЃћрЃЏрЃљрЃА. рЃАрЃљрЃГрЃўрЃарЃЮрЃљ:

- рЃАрЃгрЃЮрЃарЃў **рЃЎрЃЮрЃЮрЃарЃЊрЃўрЃюрЃљрЃбрЃБрЃџрЃў рЃАрЃўрЃАрЃбрЃћрЃЏрЃўрЃА рЃЏрЃўрЃюрЃўрЃГрЃћрЃЉрЃљ** GIS-рЃерЃў
- рЃљрЃю **Reference Points** РђЊ рЃфрЃюрЃЮрЃЉрЃљрЃЊрЃў рЃгрЃћрЃарЃбрЃўрЃџрЃћрЃЉрЃўрЃА рЃЊрЃљрЃ«рЃЏрЃљрЃарЃћрЃЉрЃўрЃЌ рЃњрЃћрЃЮрЃарЃћрЃцрЃћрЃарЃћрЃюрЃфрЃўрЃарЃћрЃЉрЃљ

## ­Ъћ╣ рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃА рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃћрЃЉрЃў GIS-рЃерЃў

1. **рЃљрЃарЃЦрЃўрЃбрЃћрЃЦрЃбрЃБрЃарЃБрЃџрЃў рЃюрЃљрЃ«рЃљрЃќрЃћрЃЉрЃўрЃА рЃўрЃюрЃбрЃћрЃњрЃарЃљрЃфрЃўрЃљ рЃБрЃарЃЉрЃљрЃюрЃБрЃџ GIS рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃЌрЃљрЃю**
2. **рЃЦрЃљрЃџрЃљрЃЦрЃЌрЃЏрЃерЃћрЃюрЃћрЃЉрЃџрЃЮрЃЉрЃўрЃА рЃарЃБрЃЎрЃћрЃЉрЃќрЃћ рЃљрЃарЃАрЃћрЃЉрЃБрЃџрЃў рЃўрЃюрЃцрЃарЃљрЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃўрЃА рЃЊрЃљрЃбрЃљрЃюрЃљ**
3. **рЃўрЃюрЃЪрЃўрЃюрЃарЃБрЃџрЃў рЃЦрЃАрЃћрЃџрЃћрЃЉрЃўрЃА (рЃгрЃДрЃљрЃџрЃў, рЃЊрЃћрЃюрЃў, рЃњрЃљрЃќрЃў) рЃЋрЃўрЃќрЃБрЃљрЃџрЃўрЃќрЃљрЃфрЃўрЃљ**
4. **рЃњрЃћрЃЮрЃЊрЃћрЃќрЃўрЃБрЃарЃў рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃА рЃљрЃюрЃљрЃџрЃўрЃќрЃў рЃАрЃ«рЃЋрЃљ рЃАрЃўрЃЋрЃарЃфрЃўрЃЌ рЃцрЃћрЃюрЃћрЃЉрЃЌрЃљрЃю рЃћрЃарЃЌрЃљрЃЊ**

## ­Ъћ╣ рЃЎрЃЮрЃюрЃЋрЃћрЃарЃбрЃљрЃфрЃўрЃљ CAD Рєњ GIS

| рЃўрЃюрЃАрЃбрЃарЃБрЃЏрЃћрЃюрЃбрЃў      | рЃљрЃдрЃгрЃћрЃарЃљ                                  |
|------------------|------------------------------------------|
| `ogr2ogr`         | рЃЎрЃЮрЃюрЃЋрЃћрЃарЃбрЃљрЃфрЃўрЃљ DXF Рєњ Shapefile рЃљрЃю GeoJSON   |
| QGIS Processing  | DXF layers Рєњ Shapefile                   |
| ArcGIS Toolboxes | CAD to GDB/Feature Class                 |

## ­Ъћ╣ рЃЎрЃљрЃарЃњрЃў рЃърЃарЃљрЃЦрЃбрЃўрЃЎрЃљ

- рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЌ рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ **DXF (R2007 рЃљрЃю рЃЊрЃљрЃЉрЃљрЃџрЃў рЃЋрЃћрЃарЃАрЃўрЃљ)** рЃцрЃЮрЃарЃЏрЃљрЃбрЃў QGIS-рЃАрЃЌрЃЋрЃўрЃА.
- рЃЊрЃљрЃарЃгрЃЏрЃБрЃюрЃЊрЃўрЃЌ, рЃарЃЮрЃЏ CAD рЃюрЃљрЃ«рЃљрЃќрЃў **рЃќрЃБрЃАрЃбрЃў рЃЎрЃЮрЃЮрЃарЃЊрЃўрЃюрЃљрЃбрЃћрЃЉрЃўрЃЌ** рЃљрЃарЃўрЃА рЃљрЃњрЃћрЃЉрЃБрЃџрЃў.
- рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЌ рЃцрЃћрЃюрЃўрЃА рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃљ (Layer names), рЃарЃљрЃЌрЃљ GIS-рЃерЃў рЃЏрЃљрЃарЃбрЃўрЃЋрЃљрЃЊ рЃЏрЃЮрЃ«рЃЊрЃћрЃА рЃЏрЃљрЃЌрЃў рЃЎрЃџрЃљрЃАрЃўрЃцрЃўрЃЎрЃљрЃфрЃўрЃљ.
