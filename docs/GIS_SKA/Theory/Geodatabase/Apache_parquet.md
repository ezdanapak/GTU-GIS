
# ­ЪЊд Apache Parquet РђЊ рЃЎрЃЮрЃџрЃЮрЃюрЃБрЃарЃў рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃцрЃЮрЃарЃЏрЃљрЃбрЃў

**Apache Parquet** рЃљрЃарЃўрЃА рЃдрЃўрЃљ, рЃЎрЃЮрЃџрЃЮрЃюрЃБрЃарЃў (рЃАрЃЋрЃћрЃбрЃБрЃарЃў) рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃцрЃЮрЃарЃЏрЃљрЃбрЃў, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃЮрЃърЃбрЃўрЃЏрЃўрЃќрЃўрЃарЃћрЃЉрЃБрЃџрЃўрЃљ рЃЊрЃўрЃЊрЃЏрЃљрЃАрЃерЃбрЃљрЃЉрЃўрЃљрЃюрЃў рЃљрЃюрЃљрЃџрЃўрЃбрЃўрЃЎрЃБрЃарЃў рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃА рЃћрЃцрЃћрЃЦрЃбрЃБрЃарЃљрЃЊ рЃерЃћрЃАрЃљрЃюрЃљрЃ«рЃљрЃЊ рЃЊрЃљ рЃњрЃљрЃЊрЃљрЃАрЃљрЃфрЃћрЃЏрЃљрЃЊ.

---

## ­ЪћЇ рЃарЃљрЃбрЃЮрЃЏ Parquet?

- **рЃЎрЃЮрЃџрЃЮрЃюрЃБрЃарЃў рЃерЃћрЃюрЃљрЃ«рЃЋрЃљ** РђЊ рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃў рЃўрЃюрЃљрЃ«рЃћрЃЉрЃљ рЃАрЃЋрЃћрЃбрЃћрЃЉрЃўрЃА рЃЏрЃўрЃ«рЃћрЃЊрЃЋрЃўрЃЌ рЃЊрЃљ рЃљрЃарЃљ рЃЏрЃгрЃЎрЃарЃўрЃЋрЃћрЃЉрЃљрЃЊ, рЃарЃљрЃф рЃљрЃЕрЃЦрЃљрЃарЃћрЃЉрЃА рЃљрЃюрЃљрЃџрЃўрЃбрЃўрЃЎрЃБрЃа рЃерЃћрЃЎрЃўрЃЌрЃ«рЃЋрЃћрЃЉрЃА.
- **рЃЏрЃљрЃдрЃљрЃџрЃў рЃЎрЃЮрЃЏрЃърЃарЃћрЃАрЃўрЃљ** РђЊ рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃў рЃБрЃЎрЃћрЃЌрЃћрЃАрЃљрЃЊ рЃўрЃЎрЃБрЃЏрЃерЃћрЃЉрЃљ, рЃарЃљрЃЊрЃњрЃљрЃю рЃћрЃарЃЌ рЃАрЃЋрЃћрЃбрЃерЃў рЃћрЃарЃЌрЃў рЃЊрЃљ рЃўрЃњрЃўрЃЋрЃћ рЃбрЃўрЃърЃўрЃА рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃљ.
- **рЃЏрЃарЃљрЃЋрЃљрЃџрЃћрЃюрЃЮрЃЋрЃљрЃюрЃў рЃЏрЃ«рЃљрЃарЃЊрЃљрЃГрЃћрЃарЃљ** РђЊ рЃерЃћрЃАрЃљрЃФрЃџрЃћрЃЉрЃћрЃџрЃўрЃљ рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ Python (pandas, pyarrow), R, Java, Spark рЃЊрЃљ рЃАрЃ«рЃЋрЃљ рЃбрЃћрЃЦрЃюрЃЮрЃџрЃЮрЃњрЃўрЃћрЃЉрЃЌрЃљрЃю.
- **рЃерЃћрЃћрЃАрЃљрЃЉрЃљрЃЏрЃћрЃЉрЃљ Hadoop-рЃўрЃА рЃћрЃЎрЃЮрЃАрЃўрЃАрЃбрЃћрЃЏрЃљрЃА** РђЊ рЃЌрЃљрЃЋрЃАрЃћрЃЉрЃљрЃЊрЃўрЃљ Hive, Impala, Drill, Spark рЃЊрЃљ рЃАрЃ«рЃЋрЃћрЃЉрЃЌрЃљрЃю.

---

## ­ЪД▒ рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃљ

Parquet рЃцрЃљрЃўрЃџрЃў рЃерЃћрЃЊрЃњрЃћрЃЉрЃљ:

- **Row Group-рЃћрЃЉрЃўрЃАрЃњрЃљрЃю** РђЊ рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃюрЃљрЃгрЃДрЃЋрЃћрЃбрЃћрЃЉрЃў, рЃарЃЮрЃЏрЃџрЃћрЃЉрЃўрЃф рЃерЃћрЃўрЃфрЃљрЃЋрЃћрЃю рЃарЃўрЃњрЃћрЃЉрЃА.
- **Column Chunk-рЃћрЃЉрЃўрЃАрЃњрЃљрЃю** РђЊ рЃЌрЃўрЃЌрЃЮрЃћрЃБрЃџрЃў рЃАрЃЋрЃћрЃбрЃўрЃА рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃў Row Group-рЃўрЃА рЃерЃўрЃњрЃюрЃўрЃЌ.
- **Page-рЃћрЃЉрЃўрЃАрЃњрЃљрЃю** РђЊ Column Chunk рЃЊрЃљрЃДрЃЮрЃцрЃўрЃџрЃўрЃљ рЃњрЃЋрЃћрЃарЃЊрЃћрЃЉрЃљрЃЊ. рЃЌрЃўрЃЌрЃЮ рЃњрЃЋрЃћрЃарЃЊрЃў рЃерЃћрЃўрЃфрЃљрЃЋрЃА рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃА рЃљрЃю рЃЏрЃћрЃбрЃљрЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃА.

---

## ­ЪЊѕ рЃБрЃърЃўрЃарЃљрЃбрЃћрЃАрЃЮрЃЉрЃћрЃЉрЃў

| рЃБрЃърЃўрЃарЃљрЃбрЃћрЃАрЃЮрЃЉрЃљ | рЃљрЃдрЃгрЃћрЃарЃљ |
|-------------|--------|
| ­Ъџђ рЃАрЃгрЃарЃљрЃцрЃў рЃЎрЃўрЃЌрЃ«рЃЋрЃљ | рЃњрЃљрЃюрЃАрЃљрЃЎрЃБрЃЌрЃарЃћрЃЉрЃўрЃЌ рЃћрЃцрЃћрЃЦрЃбрЃБрЃарЃўрЃљ рЃўрЃЏ рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃљрЃерЃў, рЃарЃЮрЃфрЃљ рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃњрЃљрЃарЃЎрЃЋрЃћрЃБрЃџрЃў рЃАрЃЋрЃћрЃбрЃћрЃЉрЃўрЃљ рЃАрЃљрЃГрЃўрЃарЃЮ |
| ­ЪњЙ рЃАрЃўрЃЋрЃарЃфрЃўрЃА рЃћрЃЎрЃЮрЃюрЃЮрЃЏрЃўрЃљ | рЃЎрЃљрЃарЃњрЃљрЃЊ рЃўрЃЎрЃБрЃЏрЃерЃћрЃЉрЃљ рЃЊрЃљ рЃљрЃЏрЃфрЃўрЃарЃћрЃЉрЃА рЃАрЃљрЃфрЃљрЃЋрЃќрЃћ рЃЊрЃљрЃбрЃЋрЃўрЃарЃЌрЃЋрЃљрЃА |
| ­ЪћЂ рЃАрЃЎрЃљрЃџрЃўрЃарЃћрЃЉрЃљрЃЊрЃЮрЃЉрЃљ | рЃљрЃЊрЃЋрЃўрЃџрЃљрЃЊ рЃљрЃЊрЃљрЃърЃбрЃўрЃарЃЊрЃћрЃЉрЃљ рЃЊрЃўрЃЊ рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃюрЃљрЃЎрЃљрЃЊрЃћрЃЉрЃќрЃћ |
| ­Ъћї рЃћрЃЎрЃЮрЃАрЃўрЃАрЃбрЃћрЃЏрЃљрЃАрЃЌрЃљрЃю рЃўрЃюрЃбрЃћрЃњрЃарЃљрЃфрЃўрЃљ | рЃЎрЃљрЃарЃњрЃљрЃЊ рЃЏрЃБрЃерЃљрЃЮрЃЉрЃА Spark, Hadoop, Hive, Presto рЃЊрЃљ рЃАрЃ«рЃЋрЃћрЃЉрЃЌрЃљрЃю |

---

## ­ЪДф рЃарЃЮрЃњрЃЮрЃа рЃњрЃљрЃЏрЃЮрЃЋрЃўрЃДрЃћрЃюрЃЮрЃЌ Python-рЃерЃў?

```python
import pandas as pd

# CSV-рЃЊрЃљрЃю Parquet рЃцрЃЮрЃарЃЏрЃљрЃбрЃерЃў рЃњрЃљрЃарЃЊрЃљрЃЦрЃЏрЃюрЃљ
df = pd.read_csv("data.csv")
df.to_parquet("data.parquet")

# Parquet рЃцрЃљрЃўрЃџрЃўрЃА рЃгрЃљрЃЎрЃўрЃЌрЃ«рЃЋрЃљ
df_parquet = pd.read_parquet("data.parquet")
```

> ­ЪњА рЃерЃћрЃюрЃўрЃерЃЋрЃюрЃљ: рЃАрЃљрЃГрЃўрЃарЃЮрЃљ `pyarrow` рЃљрЃю `fastparquet` рЃЉрЃўрЃЉрЃџрЃўрЃЮрЃЌрЃћрЃЎрЃљ.

---

## ­Ъєџ Parquet vs CSV

| рЃЏрЃљрЃ«рЃљрЃАрЃўрЃљрЃЌрЃћрЃЉрЃћрЃџрЃў     | Parquet     | CSV         |
|-------------------|-------------|-------------|
| рЃерЃћрЃюрЃљрЃ«рЃЋрЃўрЃА рЃбрЃўрЃърЃў      | рЃЎрЃЮрЃџрЃЮрЃюрЃБрЃарЃў     | рЃЏрЃгрЃЎрЃарЃўрЃЋрЃў      |
| рЃцрЃљрЃўрЃџрЃўрЃА рЃќрЃЮрЃЏрЃљ       | рЃЏрЃфрЃўрЃарЃћ        | рЃЊрЃўрЃЊрЃў         |
| рЃгрЃљрЃЎрЃўрЃЌрЃ«рЃЋрЃўрЃА рЃАрЃўрЃЕрЃЦрЃљрЃарЃћ | рЃЏрЃљрЃдрЃљрЃџрЃў       | рЃЊрЃљрЃЉрЃљрЃџрЃў       |
| рЃЏрЃћрЃбрЃљрЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃў     | рЃЎрЃў           | рЃљрЃарЃљ          |

---

## ­Ъїљ рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃА рЃАрЃцрЃћрЃарЃЮрЃћрЃЉрЃў

- рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃљрЃюрЃљрЃџрЃўрЃбрЃўрЃЎрЃљ рЃЊрЃљ BI
- рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃерЃћрЃюрЃљрЃ«рЃЋрЃљ рЃЊрЃўрЃЊ рЃЏрЃЮрЃфрЃБрЃџрЃЮрЃЉрЃљрЃќрЃћ (Data Lake)
- ETL рЃърЃарЃЮрЃфрЃћрЃАрЃћрЃЉрЃў
- Machine Learning рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃА рЃгрЃўрЃюрЃљрЃАрЃгрЃљрЃарЃў рЃЊрЃљрЃЏрЃБрЃерЃљрЃЋрЃћрЃЉрЃљ

## РюЁ рЃерЃћрЃ»рЃљрЃЏрЃћрЃЉрЃљ

Apache Parquet рЃгрЃљрЃарЃЏрЃЮрЃљрЃЊрЃњрЃћрЃюрЃА рЃЌрЃљрЃюрЃљрЃЏрЃћрЃЊрЃарЃЮрЃЋрЃћ, рЃћрЃцрЃћрЃЦрЃбрЃБрЃа рЃЊрЃљ рЃАрЃгрЃарЃљрЃц рЃцрЃљрЃўрЃџрЃўрЃА рЃцрЃЮрЃарЃЏрЃљрЃбрЃА рЃЊрЃўрЃЊрЃў рЃЏрЃЮрЃфрЃБрЃџрЃЮрЃЉрЃўрЃА рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃљрЃюрЃљрЃџрЃўрЃќрЃўрЃАрЃЌрЃЋрЃўрЃА. рЃўрЃА рЃњрЃљрЃюрЃАрЃљрЃЎрЃБрЃЌрЃарЃћрЃЉрЃўрЃЌ рЃњрЃљрЃЏрЃЮрЃАрЃљрЃЊрЃћрЃњрЃўрЃљ рЃЏрЃљрЃерЃўрЃю, рЃарЃЮрЃфрЃљ рЃАрЃљрЃГрЃўрЃарЃЮрЃљ рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃњрЃљрЃарЃЎрЃЋрЃћрЃБрЃџрЃў рЃАрЃЋрЃћрЃбрЃћрЃЉрЃўрЃА рЃгрЃљрЃЎрЃўрЃЌрЃ«рЃЋрЃљ рЃљрЃю рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃА рЃЎрЃЮрЃЏрЃърЃљрЃЦрЃбрЃБрЃарЃў рЃерЃћрЃюрЃљрЃ«рЃЋрЃљ.
