
# GeoJSON-рЃўрЃА рЃцрЃЮрЃарЃЏрЃљрЃбрЃћрЃЉрЃў QGIS-рЃЊрЃљрЃю рЃЊрЃљ ArcGIS-рЃЊрЃљрЃю

GeoJSON рЃљрЃарЃўрЃА рЃАрЃўрЃЋрЃарЃфрЃБрЃџрЃў рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃА рЃњрЃљрЃЊрЃљрЃфрЃЋрЃџрЃўрЃА рЃцрЃЮрЃарЃЏрЃљрЃбрЃў, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃЊрЃљрЃцрЃБрЃФрЃюрЃћрЃЉрЃБрЃџрЃўрЃљ JSON рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃљрЃќрЃћ рЃЊрЃљ рЃцрЃљрЃарЃЌрЃЮрЃЊ рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ рЃЋрЃћрЃЉ рЃарЃБрЃЎрЃћрЃЉрЃАрЃљ рЃЊрЃљ GIS рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃћрЃЉрЃерЃў.

## ­ЪЊї 1. QGIS РђЊ Newline Delimited GeoJSON (`.geojsonl` рЃљрЃю `.ndgeojson`)

- **рЃљрЃдрЃгрЃћрЃарЃљ**: рЃЌрЃўрЃЌрЃЮрЃћрЃБрЃџрЃў рЃ«рЃљрЃќрЃќрЃћ рЃЏрЃЮрЃЌрЃљрЃЋрЃАрЃћрЃЉрЃБрЃџрЃўрЃљ рЃћрЃарЃЌрЃў GeoJSON рЃЮрЃЉрЃўрЃћрЃЦрЃбрЃў (Feature).
- **рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ**: рЃАрЃљрЃАрЃљрЃарЃњрЃћрЃЉрЃџрЃЮрЃљ рЃЊрЃўрЃЊрЃў рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃА рЃюрЃљрЃЎрЃљрЃЊрЃўрЃА (streaming) рЃљрЃю рЃърЃљрЃарЃљрЃџрЃћрЃџрЃБрЃарЃў рЃЊрЃљрЃЏрЃБрЃерЃљрЃЋрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА.
- **QGIS рЃћрЃЦрЃАрЃърЃЮрЃарЃбрЃў**: рЃћрЃЦрЃАрЃърЃЮрЃарЃбрЃўрЃАрЃљрЃА рЃљрЃўрЃарЃЕрЃўрЃћрЃЌ _Newline delimited GeoJSON_.
- **рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў**:

```
{"type":"Feature","geometry":{"type":"Point","coordinates":[41.7,44.8]},"properties":{"name":"рЃЌрЃЉрЃўрЃџрЃўрЃАрЃў"}}
{"type":"Feature","geometry":{"type":"Point","coordinates":[42.2,42.3]},"properties":{"name":"рЃЦрЃБрЃЌрЃљрЃўрЃАрЃў"}}
```

## ­ЪЊї 2. QGIS РђЊ рЃАрЃбрЃљрЃюрЃЊрЃљрЃарЃбрЃБрЃџрЃў GeoJSON

- **рЃљрЃдрЃгрЃћрЃарЃљ**: рЃЏрЃЌрЃћрЃџрЃў FeatureCollection рЃћрЃарЃЌрЃўрЃљрЃю JSON рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃљрЃерЃўрЃљ рЃЏрЃЮрЃЌрЃљрЃЋрЃАрЃћрЃЉрЃБрЃџрЃў.
- **рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ**: рЃќрЃЮрЃњрЃљрЃЊ рЃЏрЃўрЃќрЃюрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА, рЃЋрЃћрЃЉ рЃарЃБрЃЎрЃћрЃЉрЃќрЃћ рЃњрЃљрЃЏрЃЮрЃАрЃљрЃбрЃљрЃюрЃљрЃЊ.
- **QGIS рЃћрЃЦрЃАрЃърЃЮрЃарЃбрЃў**: рЃљрЃўрЃарЃЕрЃўрЃћрЃЌ _GeoJSON_.
- **рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў**:

```json
{
  "type": "FeatureCollection",
  "features": [
    {
      "type": "Feature",
      "geometry": {
        "type": "Point",
        "coordinates": [41.7, 44.8]
      },
      "properties": {
        "name": "рЃЌрЃЉрЃўрЃџрЃўрЃАрЃў"
      }
    },
    {
      "type": "Feature",
      "geometry": {
        "type": "Point",
        "coordinates": [42.2, 42.3]
      },
      "properties": {
        "name": "рЃЦрЃБрЃЌрЃљрЃўрЃАрЃў"
      }
    }
  ]
}
```

## ­ЪЊї 3. ArcGIS РђЊ Formatted GeoJSON

- **рЃљрЃдрЃгрЃћрЃарЃљ**: рЃАрЃбрЃљрЃюрЃЊрЃљрЃарЃбрЃБрЃџрЃљрЃЊ рЃцрЃЮрЃарЃЏрЃљрЃбрЃўрЃарЃћрЃЉрЃБрЃџрЃў (рЃЊрЃљрЃЏрЃБрЃерЃљрЃЋрЃћрЃЉрЃБрЃџрЃў) JSON рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃљ, рЃљрЃЊрЃЋрЃўрЃџрЃљрЃЊ рЃгрЃљрЃАрЃљрЃЎрЃўрЃЌрЃ«рЃў рЃљрЃЊрЃљрЃЏрЃўрЃљрЃюрЃўрЃАрЃЌрЃЋрЃўрЃА.
- **ArcGIS Pro**: рЃћрЃЦрЃАрЃърЃЮрЃарЃбрЃўрЃАрЃљрЃА рЃерЃћрЃАрЃљрЃФрЃџрЃћрЃЉрЃћрЃџрЃўрЃљ JSON рЃцрЃЮрЃарЃЏрЃљрЃбрЃерЃў рЃерЃћрЃюрЃљрЃ«рЃЋрЃљ рЃЊрЃљ рЃњрЃљрЃцрЃЮрЃарЃЏрЃћрЃЉрЃљ.
- **рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ**: рЃЎрЃЮрЃЊрЃерЃў рЃЕрЃљрЃАрЃљрЃгрЃћрЃарЃљрЃЊ, рЃњрЃљрЃЊрЃљрЃфрЃћрЃЏрЃўрЃАрЃЌрЃЋрЃўрЃА API-рЃА рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃўрЃЌ.

```json
{
  "type": "FeatureCollection",
  "features": [
    {
      "type": "Feature",
      "geometry": {
        "type": "Point",
        "coordinates": [43.0, 41.9]
      },
      "properties": {
        "name": "рЃЉрЃљрЃЌрЃБрЃЏрЃў"
      }
    }
  ]
}
```

## ­ЪЊї 4. ArcGIS РђЊ Non-formatted (Minified) GeoJSON

- **рЃљрЃдрЃгрЃћрЃарЃљ**: рЃДрЃЋрЃћрЃџрЃљ рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃљ рЃћрЃарЃЌ рЃ«рЃљрЃќрЃќрЃћрЃљ РђЊ рЃЎрЃЮрЃЏрЃърЃљрЃЦрЃбрЃБрЃарЃў рЃцрЃЮрЃарЃЏрЃљрЃбрЃўрЃљ.
- **рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ**: рЃарЃЮрЃЊрЃћрЃАрЃљрЃф рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЋрЃљрЃюрЃўрЃљ рЃцрЃљрЃўрЃџрЃўрЃА рЃќрЃЮрЃЏрЃљ рЃљрЃю рЃњрЃљрЃЊрЃљрЃфрЃћрЃЏрЃўрЃА рЃАрЃўрЃЕрЃЦрЃљрЃарЃћ.
- **рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў**:

```
{"type":"FeatureCollection","features":[{"type":"Feature","geometry":{"type":"Point","coordinates":[43.0,41.9]},"properties":{"name":"рЃЉрЃљрЃЌрЃБрЃЏрЃў"}}]}
```
