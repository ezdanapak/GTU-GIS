# рЃарЃБрЃЎрЃўрЃА рЃЎрЃЮрЃЏрЃърЃЮрЃќрЃўрЃфрЃўрЃљ (Map Composition)

рЃарЃБрЃЎрЃўрЃА рЃЎрЃЮрЃЏрЃърЃЮрЃќрЃўрЃфрЃўрЃљ рЃљрЃарЃўрЃА рЃарЃБрЃЎрЃўрЃА рЃДрЃЋрЃћрЃџрЃљ рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃўрЃА рЃАрЃгрЃЮрЃарЃљрЃЊ рЃњрЃљрЃюрЃџрЃљрЃњрЃћрЃЉрЃљ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃБрЃќрЃарЃБрЃюрЃЋрЃћрЃџрЃДрЃЮрЃцрЃА рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃўрЃА рЃћрЃцрЃћрЃЦрЃбрЃБрЃарЃљрЃЊ рЃЊрЃљ рЃЏрЃЎрЃљрЃцрЃўрЃЮрЃЊ рЃњрЃљрЃЊрЃљрЃфрЃћрЃЏрЃљрЃА. рЃАрЃгрЃЮрЃарЃљрЃЊ рЃЊрЃљрЃърЃарЃЮрЃћрЃЦрЃбрЃћрЃЉрЃБрЃџрЃў рЃарЃБрЃЎрЃљ рЃљрЃБрЃЏрЃ»рЃЮрЃЉрЃћрЃАрЃћрЃЉрЃА рЃљрЃдрЃЦрЃЏрЃљрЃА рЃЊрЃљ рЃћрЃ«рЃЏрЃљрЃарЃћрЃЉрЃљ рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃћрЃџрЃА рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃА рЃАрЃгрЃЮрЃа рЃўрЃюрЃбрЃћрЃарЃърЃарЃћрЃбрЃљрЃфрЃўрЃљрЃерЃў.

---

## ­ЪДЕ рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃў рЃЎрЃЮрЃЏрЃърЃЮрЃќрЃўрЃфрЃўрЃБрЃарЃў рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃћрЃЉрЃў

| рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃў           | рЃљрЃдрЃгрЃћрЃарЃљ |
|--------------------|--------|
| **рЃАрЃљрЃЌрЃљрЃБрЃарЃў** (Title)     | рЃ«рЃљрЃюрЃЏрЃЮрЃЎрЃџрЃћ рЃЊрЃљ рЃќрЃБрЃАрЃбрЃў рЃљрЃдрЃгрЃћрЃарЃљ, рЃарЃљрЃф рЃњрЃЋрЃўрЃЕрЃЋрЃћрЃюрЃћрЃЉрЃА рЃарЃБрЃЎрЃўрЃА рЃерЃўрЃюрЃљрЃљрЃарЃАрЃА. |
| **рЃџрЃћрЃњрЃћрЃюрЃЊрЃљ** (Legend)     | рЃАрЃўрЃЏрЃЉрЃЮрЃџрЃЮрЃћрЃЉрЃўрЃА рЃЊрЃљ рЃцрЃћрЃарЃћрЃЉрЃўрЃА рЃњрЃљрЃюрЃЏрЃљрЃарЃбрЃћрЃЉрЃљ. |
| **рЃЏрЃљрЃАрЃерЃбрЃљрЃЉрЃў** (Scale)     | рЃљрЃЕрЃЋрЃћрЃюрЃћрЃЉрЃА рЃарЃБрЃЎрЃљрЃќрЃћ рЃЏрЃљрЃюрЃФрЃўрЃџрЃЌрЃљ рЃерЃћрЃАрЃљрЃЉрЃљрЃЏрЃўрЃАрЃЮрЃЉрЃљрЃА рЃарЃћрЃљрЃџрЃБрЃа рЃАрЃўрЃЋрЃарЃфрЃћрЃАрЃЌрЃљрЃю. |
| **рЃЕрЃарЃЊрЃўрЃџрЃЮрЃћрЃЌрЃўрЃА рЃўрЃАрЃљрЃарЃў** (North Arrow) | рЃњрЃЋрЃћрЃ«рЃЏрЃљрЃарЃћрЃЉрЃљ рЃЏрЃўрЃЏрЃљрЃарЃЌрЃБрЃџрЃћрЃЉрЃўрЃА рЃњрЃљрЃарЃЎрЃЋрЃћрЃЋрЃљрЃерЃў. |
| **рЃАрЃљрЃЎрЃЮрЃЮрЃарЃЊрЃўрЃюрЃљрЃбрЃЮ рЃЉрЃљрЃЊрЃћ** (Graticule/Grid) | рЃЎрЃЮрЃЮрЃарЃЊрЃўрЃюрЃљрЃбрЃћрЃЉрЃўрЃА рЃЉрЃљрЃЊрЃћ рЃљрЃю рЃњрЃћрЃЮрЃњрЃарЃљрЃцрЃўрЃБрЃџрЃў рЃњрЃарЃљрЃбрЃўрЃЎрЃБрЃџрЃљ рЃарЃБрЃЎрЃљрЃќрЃћ. |
| **рЃгрЃДрЃљрЃарЃЮ** (Data Source) | рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃгрЃљрЃарЃЏрЃЮрЃЏрЃљрЃЋрЃџрЃЮрЃЉрЃўрЃА рЃљрЃдрЃюрЃўрЃерЃЋрЃюрЃљ. |
| **рЃљрЃЋрЃбрЃЮрЃарЃў рЃЊрЃљ рЃЌрЃљрЃарЃўрЃдрЃў** | рЃарЃБрЃЎрЃўрЃА рЃљрЃЋрЃбрЃЮрЃарЃўрЃАрЃљ рЃЊрЃљ рЃерЃћрЃЦрЃЏрЃюрЃўрЃА рЃЌрЃљрЃарЃўрЃдрЃўрЃА рЃЏрЃўрЃЌрЃўрЃЌрЃћрЃЉрЃљ. |

---

## ­Ъј» рЃарЃБрЃЎрЃўрЃА рЃЎрЃЮрЃЏрЃърЃЮрЃќрЃўрЃфрЃўрЃўрЃА рЃЏрЃўрЃќрЃљрЃюрЃў

- рЃЋрЃўрЃќрЃБрЃљрЃџрЃБрЃарЃљрЃЊ рЃгрЃљрЃарЃљрЃЊрЃњрЃўрЃюрЃЮрЃА рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃў рЃЏрЃЎрЃљрЃцрЃўрЃЮрЃЊ рЃЊрЃљ рЃџрЃЮрЃњрЃўрЃЎрЃБрЃарЃљрЃЊ
- рЃњрЃљрЃќрЃљрЃарЃЊрЃЮрЃА рЃарЃБрЃЎрЃўрЃА рЃгрЃљрЃЎрЃўрЃЌрЃ«рЃЋрЃљрЃЊрЃЮрЃЉрЃљ
- рЃЏрЃўрЃАрЃфрЃћрЃА рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃћрЃџрЃА рЃЎрЃЮрЃюрЃбрЃћрЃЦрЃАрЃбрЃў, рЃўрЃюрЃбрЃћрЃарЃърЃарЃћрЃбрЃљрЃфрЃўрЃўрЃАрЃЌрЃЋрЃўрЃА рЃАрЃљрЃГрЃўрЃарЃЮ рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃћрЃЉрЃўрЃЌ
- рЃБрЃќрЃарЃБрЃюрЃЋрЃћрЃџрЃДрЃЮрЃА рЃърЃарЃЮрЃцрЃћрЃАрЃўрЃЮрЃюрЃљрЃџрЃБрЃарЃў рЃЊрЃљ рЃћрЃАрЃЌрЃћрЃбрЃўрЃЎрЃБрЃарЃў рЃерЃЌрЃљрЃЉрЃћрЃГрЃЊрЃўрЃџрЃћрЃЉрЃљ

---

## ­ЪЏа№ИЈ рЃарЃБрЃЎрЃўрЃА рЃЎрЃЮрЃЏрЃърЃЮрЃќрЃўрЃфрЃўрЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃљ QGIS-рЃерЃў

1. рЃЏрЃћрЃюрЃўрЃБрЃЊрЃљрЃю рЃљрЃўрЃарЃЕрЃўрЃћ **Project Рєњ New Print Layout**.
2. рЃЊрЃљрЃљрЃарЃЦрЃЋрЃў рЃАрЃљрЃ«рЃћрЃџрЃў рЃЊрЃљ рЃЊрЃљрЃљрЃЊрЃљрЃАрЃбрЃБрЃарЃћ.
3. рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћ рЃўрЃюрЃАрЃбрЃарЃБрЃЏрЃћрЃюрЃбрЃћрЃЉрЃў рЃЎрЃЮрЃЏрЃърЃЮрЃќрЃўрЃфрЃўрЃљрЃерЃў рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃћрЃЉрЃўрЃА рЃЊрЃљрЃАрЃљрЃЏрЃљрЃбрЃћрЃЉрЃџрЃљрЃЊ:
   - `Add Map` РђЊ рЃарЃБрЃЎрЃўрЃА рЃЕрЃљрЃАрЃљрЃАрЃЏрЃћрЃџрЃљрЃЊ
   - `Add Legend`, `Add Scale Bar`, `Add North Arrow` РђЊ рЃерЃћрЃАрЃљрЃЉрЃљрЃЏрЃўрЃАрЃў рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА
   - `Add Label` РђЊ рЃбрЃћрЃЦрЃАрЃбрЃўрЃА рЃЕрЃљрЃАрЃљрЃЏрЃљрЃбрЃћрЃЉрЃџрЃљрЃЊ
4. рЃЊрЃљрЃљрЃарЃћрЃњрЃБрЃџрЃўрЃарЃћ рЃќрЃЮрЃЏрЃљ, рЃцрЃћрЃарЃћрЃЉрЃў, рЃцрЃЮрЃюрЃбрЃў рЃЊрЃљ рЃњрЃљрЃюрЃџрЃљрЃњрЃћрЃЉрЃљ.
5. рЃЊрЃљрЃљрЃЉрЃћрЃГрЃЊрЃћ рЃљрЃю рЃерЃћрЃўрЃюрЃљрЃ«рЃћ PDF/PNG рЃцрЃЮрЃарЃЏрЃљрЃбрЃерЃў.

---

## РюЁ рЃЎрЃљрЃарЃњрЃў рЃарЃБрЃЎрЃўрЃА рЃЎрЃЮрЃЏрЃърЃЮрЃќрЃўрЃфрЃўрЃўрЃА рЃгрЃћрЃАрЃћрЃЉрЃў

- рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћ рЃЎрЃЮрЃюрЃбрЃарЃљрЃАрЃбрЃБрЃџрЃў рЃцрЃћрЃарЃћрЃЉрЃў рЃгрЃљрЃЎрЃўрЃЌрЃ«рЃЋрЃљрЃЊрЃЮрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА
- рЃљрЃа рЃњрЃљрЃЊрЃљрЃбрЃЋрЃўрЃарЃЌрЃЮ рЃарЃБрЃЎрЃљ рЃќрЃћрЃЊрЃЏрЃћрЃбрЃў рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃћрЃЉрЃўрЃЌ
- рЃЊрЃљрЃАрЃљрЃ«рЃћрЃџрЃћрЃЉрЃћрЃЉрЃў рЃўрЃДрЃЮрЃА рЃњрЃљрЃАрЃљрЃњрЃћрЃЉрЃў рЃЊрЃљ рЃАрЃБрЃцрЃЌрЃљ
- рЃџрЃћрЃњрЃћрЃюрЃЊрЃљ рЃЏрЃЮрЃўрЃфрЃљрЃЋрЃЊрЃћрЃА рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃўрЃЏ рЃАрЃўрЃЏрЃЉрЃЮрЃџрЃЮрЃћрЃЉрЃА, рЃарЃљрЃф рЃарЃБрЃЎрЃљрЃќрЃћрЃљ

---

## ­ЪЊї рЃарЃБрЃЎрЃўрЃА рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃЏрЃЮрЃўрЃфрЃљрЃЋрЃЊрЃћрЃА:

- рЃАрЃљрЃЌрЃљрЃБрЃарЃўрЃЌ: "рЃЌрЃЉрЃўрЃџрЃўрЃАрЃўрЃА рЃарЃљрЃўрЃЮрЃюрЃћрЃЉрЃўрЃА рЃЏрЃЮрЃАрЃљрЃ«рЃџрЃћрЃЮрЃЉрЃљ 2024"
- рЃЏрЃљрЃАрЃерЃбрЃљрЃЉрЃўрЃЌ: "1:50,000"
- рЃџрЃћрЃњрЃћрЃюрЃЊрЃўрЃЌ: рЃцрЃћрЃарЃўрЃА рЃЏрЃўрЃ«рЃћрЃЊрЃЋрЃўрЃЌ рЃЎрЃџрЃљрЃАрЃўрЃцрЃўрЃфрЃўрЃарЃћрЃЉрЃБрЃџрЃў рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃў
- рЃЕрЃарЃЊрЃўрЃџрЃЮрЃћрЃЌрЃўрЃА рЃўрЃАрЃарЃўрЃЌ рЃЏрЃљрЃарЃ»рЃЋрЃћрЃюрЃљ рЃќрЃћрЃЊрЃљ рЃЎрЃБрЃЌрЃ«рЃћрЃерЃў
- рЃгрЃДрЃљрЃарЃЮрЃА рЃЏрЃўрЃЌрЃўрЃЌрЃћрЃЉрЃўрЃЌ: "рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃў: рЃАрЃљрЃЦрЃАрЃбрЃљрЃбрЃў, 2024"

---

рЃарЃБрЃЎрЃўрЃА рЃЎрЃЮрЃЏрЃърЃЮрЃќрЃўрЃфрЃўрЃљ рЃљрЃћрЃарЃЌрЃўрЃљрЃюрЃћрЃЉрЃА рЃЎрЃљрЃарЃбрЃЮрЃњрЃарЃљрЃцрЃўрЃљрЃА, рЃЊрЃўрЃќрЃљрЃўрЃюрЃА рЃЊрЃљ рЃАрЃўрЃЋрЃарЃфрЃБрЃџ рЃљрЃюрЃљрЃџрЃўрЃќрЃА Рђћ рЃўрЃА рЃАрЃљрЃЉрЃЮрЃџрЃЮрЃЮ рЃърЃарЃЮрЃЊрЃБрЃЦрЃбрЃўрЃљ, рЃарЃЮрЃЏрЃћрЃџрЃАрЃљрЃф рЃЋрЃБрЃќрЃўрЃљрЃарЃћрЃЉрЃЌ рЃАрЃ«рЃЋрЃљ рЃљрЃЊрЃљрЃЏрЃўрЃљрЃюрЃћрЃЉрЃА. рЃљрЃЏрЃўрЃбрЃЮрЃЏ рЃЏрЃўрЃАрЃў рЃЊрЃљрЃњрЃћрЃњрЃЏрЃЋрЃљ рЃЊрЃљ рЃерЃћрЃАрЃарЃБрЃџрЃћрЃЉрЃљ рЃўрЃАрЃћрЃЋрЃћ рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЋрЃљрЃюрЃўрЃљ, рЃарЃЮрЃњрЃЮрЃарЃф рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃЊрЃљрЃЏрЃБрЃерЃљрЃЋрЃћрЃЉрЃљ.
