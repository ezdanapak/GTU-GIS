
# QGIS: рЃљрЃЉрЃАрЃЮрЃџрЃБрЃбрЃБрЃарЃў vs рЃцрЃљрЃарЃЊрЃЮрЃЉрЃўрЃЌрЃў рЃЉрЃўрЃџрЃўрЃЎрЃћрЃЉрЃў

QGIS-рЃерЃў рЃЏрЃБрЃерЃљрЃЮрЃЉрЃўрЃАрЃљрЃА рЃцрЃљрЃўрЃџрЃћрЃЉрЃўрЃА рЃЉрЃўрЃџрЃўрЃЎрЃўрЃА рЃбрЃўрЃърЃўрЃА рЃАрЃгрЃЮрЃарЃљрЃЊ рЃњрЃљрЃюрЃАрЃљрЃќрЃдрЃЋрЃарЃљ рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЋрЃљрЃюрЃўрЃљ рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃърЃЮрЃарЃбрЃљрЃЉрЃћрЃџрЃБрЃарЃЮрЃЉрЃўрЃАрЃљ рЃЊрЃљ рЃАрЃ«рЃЋрЃљ рЃЏрЃЮрЃгрЃДрЃЮрЃЉрЃўрЃџрЃЮрЃЉрЃћрЃЉрЃќрЃћ рЃАрЃгрЃЮрЃарЃљрЃЊ рЃњрЃљрЃ«рЃАрЃюрЃўрЃАрЃЌрЃЋрЃўрЃА.

QGIS рЃЮрЃцрЃўрЃфрЃўрЃљрЃџрЃБрЃарЃў [рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃљрЃфрЃўрЃљ:](https://docs.qgis.org/3.40/en/docs/user_manual/introduction/project_files.html#handling-broken-file-paths)

## ­ЪЊЂ рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃўрЃА рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў

QGIS рЃърЃарЃЮрЃћрЃЦрЃбрЃў рЃЏрЃЮрЃЌрЃљрЃЋрЃАрЃћрЃЉрЃБрЃџрЃўрЃљ рЃћрЃарЃЌ рЃАрЃљрЃћрЃарЃЌрЃЮ рЃАрЃљрЃЦрЃљрЃдрЃљрЃџрЃЊрЃћрЃерЃў рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃЌрЃљрЃю рЃћрЃарЃЌрЃљрЃЊ:

```
Chiatura_project/             Рєљ рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃў рЃАрЃљрЃЦрЃљрЃдрЃљрЃџрЃЊрЃћ
РћюРћђРћђ project.qgz              Рєљ QGIS рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃцрЃљрЃўрЃџрЃў
РћюРћђРћђ data/                    Рєљ рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃА рЃАрЃљрЃЦрЃљрЃдрЃљрЃџрЃЊрЃћ
Рћѓ   РћюРћђРћђ roads.shp            Рєљ рЃерЃарЃћ: рЃњрЃќрЃћрЃЉрЃў
Рћѓ   РћюРћђРћђ buildings.shp        Рєљ рЃерЃарЃћ: рЃерЃћрЃюрЃЮрЃЉрЃћрЃЉрЃў
Рћѓ   РћћРћђРћђ rivers.shp           Рєљ рЃерЃарЃћ: рЃЏрЃЊрЃўрЃюрЃљрЃарЃћрЃћрЃЉрЃў
```

## ­ЪЊї рЃцрЃљрЃарЃЊрЃЮрЃЉрЃўрЃЌрЃў рЃЉрЃўрЃџрЃўрЃЎрЃћрЃЉрЃў (Relative Paths)

QGIS-рЃерЃў project.qgz рЃцрЃљрЃўрЃџрЃЌрЃљрЃю рЃЏрЃўрЃЏрЃљрЃарЃЌрЃћрЃЉрЃўрЃЌ рЃцрЃљрЃўрЃџрЃўрЃА рЃЉрЃўрЃџрЃўрЃЎрЃћрЃЉрЃў рЃўрЃЦрЃюрЃћрЃЉрЃљ:

```
./data/roads.shp
./data/buildings.shp
./data/rivers.shp
```

РюЁ **рЃарЃћрЃЎрЃЮрЃЏрЃћрЃюрЃЊрЃћрЃЉрЃБрЃџрЃўрЃљ**, рЃарЃЮрЃфрЃљ рЃърЃарЃЮрЃћрЃЦрЃбрЃў рЃўрЃюрЃљрЃ«рЃћрЃЉрЃљ рЃћрЃарЃЌ рЃцрЃЮрЃџрЃЊрЃћрЃарЃерЃў рЃЊрЃљ рЃўрЃњрЃћрЃњрЃЏрЃћрЃЉрЃљ рЃЏрЃўрЃАрЃў рЃњрЃљрЃЊрЃљрЃбрЃљрЃюрЃљ рЃљрЃю рЃњрЃљрЃќрЃўрЃљрЃарЃћрЃЉрЃљ.

---

## ­ЪЊї рЃљрЃЉрЃАрЃЮрЃџрЃБрЃбрЃБрЃарЃў рЃЉрЃўрЃџрЃўрЃЎрЃћрЃЉрЃў (Absolute Paths)

рЃцрЃљрЃўрЃџрЃўрЃА рЃАрЃарЃБрЃџрЃў рЃЏрЃўрЃАрЃљрЃЏрЃљрЃарЃЌрЃў рЃЎрЃЮрЃЏрЃърЃўрЃБрЃбрЃћрЃарЃќрЃћ:

```
C:/Users/ezdanapak/Documents/Chiatura_project/data/roads.shp
C:/Users/ezdanapak/Documents/Chiatura_project/data/buildings.shp
C:/Users/ezdanapak/Documents/Chiatura_project/data/rivers.shp
```

Рџа№ИЈ **рЃљрЃа рЃљрЃарЃўрЃА рЃърЃЮрЃарЃбрЃљрЃЉрЃћрЃџрЃБрЃарЃў**. рЃўрЃЏрЃБрЃерЃљрЃЋрЃћрЃЉрЃА рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃўрЃЏрЃљрЃЋрЃћ рЃЏрЃЮрЃгрЃДрЃЮрЃЉрЃўрЃџрЃЮрЃЉрЃљрЃќрЃћ, рЃўрЃњрЃўрЃЋрЃћ рЃЉрЃўрЃџрЃўрЃЎрЃўрЃА рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃўрЃЌ.

---

## РџЎ№ИЈ рЃарЃЮрЃњрЃЮрЃа рЃерЃћрЃЋрЃфрЃЋрЃљрЃџрЃЮрЃЌ рЃЉрЃўрЃџрЃўрЃЎрЃћрЃЉрЃўрЃА рЃбрЃўрЃърЃў QGIS-рЃерЃў?

1. рЃњрЃљрЃ«рЃАрЃћрЃюрЃўрЃЌ QGIS рЃърЃарЃЮрЃћрЃЦрЃбрЃў.
2. рЃњрЃљрЃЊрЃљрЃЊрЃўрЃЌ рЃЏрЃћрЃюрЃўрЃБрЃерЃў: `Project` Рєњ `Properties` Рєњ `General`
3. рЃўрЃърЃЮрЃЋрЃћрЃЌ рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃў **Store paths**.
4. рЃљрЃўрЃарЃЕрЃўрЃћрЃЌ рЃћрЃарЃЌ-рЃћрЃарЃЌрЃў:
   - `Relative` РђЊ рЃцрЃљрЃарЃЊрЃЮрЃЉрЃўрЃЌрЃў рЃЉрЃўрЃџрЃўрЃЎрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА
   - `Absolute` РђЊ рЃљрЃЉрЃАрЃЮрЃџрЃБрЃбрЃБрЃарЃў рЃЉрЃўрЃџрЃўрЃЎрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА

---

## РюЁ рЃарЃћрЃЎрЃЮрЃЏрЃћрЃюрЃЊрЃћрЃЉрЃБрЃџрЃў рЃърЃарЃљрЃЦрЃбрЃўрЃЎрЃљ

- рЃерЃћрЃўрЃюрЃљрЃ«рЃћ QGIS рЃърЃарЃЮрЃћрЃЦрЃбрЃў рЃЊрЃљ рЃДрЃЋрЃћрЃџрЃљ рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃў рЃћрЃарЃЌ рЃАрЃљрЃћрЃарЃЌрЃЮ рЃАрЃљрЃЦрЃљрЃдрЃљрЃџрЃЊрЃћрЃерЃў.
- рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћ **рЃцрЃљрЃарЃЊрЃЮрЃЉрЃўрЃЌрЃў рЃЉрЃўрЃџрЃўрЃЎрЃћрЃЉрЃў**, рЃЌрЃБ:
  - рЃърЃарЃЮрЃћрЃЦрЃбрЃў рЃАрЃ«рЃЋрЃљрЃА рЃБрЃюрЃЊрЃљ рЃњрЃљрЃБрЃќрЃўрЃљрЃарЃЮ.
  - рЃърЃарЃЮрЃћрЃЦрЃбрЃў рЃБрЃюрЃЊрЃљ рЃњрЃљрЃЊрЃљрЃўрЃбрЃљрЃюрЃЮ рЃАрЃ«рЃЋрЃљ рЃЎрЃЮрЃЏрЃърЃўрЃБрЃбрЃћрЃарЃќрЃћ.
  - рЃўрЃДрЃћрЃюрЃћрЃЉ Git-рЃА рЃљрЃю ZIP рЃљрЃарЃЦрЃўрЃЋрЃА.
