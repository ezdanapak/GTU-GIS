
# ­ЪЊЂ QGIS рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃцрЃљрЃўрЃџрЃћрЃЉрЃў: `.qgs` vs `.qgz`

QGIS (Quantum GIS) рЃўрЃДрЃћрЃюрЃћрЃЉрЃА рЃЮрЃа рЃФрЃўрЃарЃўрЃЌрЃљрЃЊ рЃцрЃЮрЃарЃЏрЃљрЃбрЃА рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃцрЃљрЃўрЃџрЃћрЃЉрЃўрЃА рЃерЃћрЃАрЃљрЃюрЃљрЃ«рЃљрЃЊ: **`.qgs`** рЃЊрЃљ **`.qgz`**. 
рЃљрЃЏ рЃцрЃЮрЃарЃЏрЃљрЃбрЃћрЃЉрЃА рЃерЃЮрЃарЃўрЃА рЃњрЃљрЃюрЃАрЃ«рЃЋрЃљрЃЋрЃћрЃЉрЃљ рЃћрЃ«рЃћрЃЉрЃљ рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃерЃћрЃюрЃљрЃ«рЃЋрЃўрЃА рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃљрЃАрЃљ рЃЊрЃљ рЃљрЃарЃЦрЃўрЃЋрЃўрЃарЃћрЃЉрЃљрЃА. 
рЃЦрЃЋрЃћрЃЏрЃЮрЃЌ рЃЏрЃЮрЃфрЃћрЃЏрЃБрЃџрЃўрЃљ рЃЏрЃљрЃЌрЃў рЃЏрЃљрЃ«рЃљрЃАрЃўрЃљрЃЌрЃћрЃЉрЃџрЃћрЃЉрЃўрЃАрЃљ рЃЊрЃљ рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃА рЃЊрЃћрЃбрЃљрЃџрЃБрЃарЃў рЃЏрЃўрЃЏрЃЮрЃ«рЃўрЃџрЃЋрЃљ.

---

## ­Ъћ╣ `.qgs` Рђћ XML рЃцрЃЮрЃарЃЏрЃљрЃбрЃў

- **рЃљрЃдрЃгрЃћрЃарЃљ**: `.qgs` рЃљрЃарЃўрЃА **рЃАрЃбрЃљрЃюрЃЊрЃљрЃарЃбрЃБрЃџрЃў XML рЃцрЃљрЃўрЃџрЃў**, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃўрЃюрЃљрЃ«рЃљрЃЋрЃА рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃА рЃбрЃћрЃЦрЃАрЃбрЃБрЃа рЃцрЃЮрЃарЃЏрЃљрЃбрЃерЃў.
- **рЃЏрЃљрЃ«рЃљрЃАрЃўрЃљрЃЌрЃћрЃЉрЃџрЃћрЃЉрЃў**:
  - **рЃљрЃЊрЃЋрЃўрЃџрЃљрЃЊ рЃгрЃљрЃЎрЃўрЃЌрЃ«рЃЋрЃљрЃЊрЃў**: рЃерЃћрЃАрЃљрЃФрЃџрЃћрЃЉрЃћрЃџрЃўрЃљ рЃцрЃљрЃўрЃџрЃўрЃА рЃњрЃљрЃ«рЃАрЃюрЃљ рЃЊрЃљ рЃарЃћрЃЊрЃљрЃЦрЃбрЃўрЃарЃћрЃЉрЃљ рЃюрЃћрЃЉрЃўрЃАрЃЏрЃўрЃћрЃарЃў рЃбрЃћрЃЦрЃАрЃбрЃБрЃарЃў рЃарЃћрЃЊрЃљрЃЦрЃбрЃЮрЃарЃўрЃЌ (рЃЏрЃљрЃњ. Notepad++, VS Code).
  - **рЃЋрЃћрЃарЃАрЃўрЃћрЃЉрЃўрЃА рЃЎрЃЮрЃюрЃбрЃарЃЮрЃџрЃў**: рЃўрЃЊрЃћрЃљрЃџрЃБрЃарЃўрЃљ Git-рЃўрЃА рЃљрЃю рЃАрЃ«рЃЋрЃљ рЃЋрЃћрЃарЃАрЃўрЃўрЃА рЃЏрЃљрЃарЃЌрЃЋрЃўрЃА рЃАрЃўрЃАрЃбрЃћрЃЏрЃћрЃЉрЃЌрЃљрЃю рЃЏрЃБрЃерЃљрЃЮрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА, рЃарЃљрЃЊрЃњрЃљрЃю XML рЃцрЃЮрЃарЃЏрЃљрЃбрЃў рЃфрЃЋрЃџрЃўрЃџрЃћрЃЉрЃћрЃЉрЃўрЃА рЃЌрЃЋрЃљрЃџрЃДрЃБрЃарЃўрЃА рЃЊрЃћрЃЋрЃюрЃћрЃЉрЃљрЃА рЃљрЃЏрЃљрЃарЃбрЃўрЃЋрЃћрЃЉрЃА.
  - **рЃЊрЃљрЃЏрЃЮрЃБрЃЎрЃўрЃЊрЃћрЃЉрЃћрЃџрЃў рЃцрЃљрЃўрЃџрЃў**: рЃ«рЃерЃўрЃарЃљрЃЊ рЃЌрЃљрЃю рЃљрЃ«рЃџрЃљрЃЋрЃА рЃАрЃ«рЃЋрЃљ рЃЊрЃљрЃЎрЃљрЃЋрЃерЃўрЃарЃћрЃЉрЃБрЃџрЃў рЃцрЃљрЃўрЃџрЃћрЃЉрЃў/рЃцрЃЮрЃџрЃЊрЃћрЃарЃћрЃЉрЃў (рЃЏрЃљрЃњ. `project.gpkg`, рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃўрЃДрЃЮрЃА рЃАрЃ«рЃЋрЃљ рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃцрЃљрЃўрЃџрЃћрЃЉрЃў).
- **рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃА рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃљ**:
  - рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃљрЃЦрЃбрЃўрЃБрЃарЃў рЃњрЃљрЃюрЃЋрЃўрЃЌрЃљрЃарЃћрЃЉрЃљ.
  - рЃ«рЃћрЃџрЃўрЃЌ рЃарЃћрЃЊрЃљрЃЦрЃбрЃўрЃарЃћрЃЉрЃљ рЃљрЃю рЃфрЃЋрЃџрЃўрЃџрЃћрЃЉрЃћрЃЉрЃўрЃА рЃЌрЃЋрЃљрЃџрЃДрЃБрЃарЃўрЃА рЃЊрЃћрЃЋрЃюрЃћрЃЉрЃљ.

­ЪЊї **рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў**:
```bash
my_project.qgs
```

---

## ­Ъћ╣ `.qgz` Рђћ рЃерЃћрЃЎрЃБрЃЏрЃерЃБрЃџрЃў рЃърЃарЃЮрЃћрЃЦрЃбрЃў

- **рЃљрЃдрЃгрЃћрЃарЃљ**: `.qgz` рЃљрЃарЃўрЃА ZIP-рЃўрЃЌ рЃерЃћрЃЎрЃБрЃЏрЃерЃБрЃџрЃў рЃљрЃарЃЦрЃўрЃЋрЃў, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃерЃћрЃўрЃфрЃљрЃЋрЃА `.qgs` рЃцрЃљрЃўрЃџрЃАрЃљ рЃЊрЃљ рЃърЃарЃЮрЃћрЃЦрЃбрЃЌрЃљрЃю рЃЊрЃљрЃЎрЃљрЃЋрЃерЃўрЃарЃћрЃЉрЃБрЃџ рЃАрЃ«рЃЋрЃљ рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃА.
- **рЃЏрЃљрЃ«рЃљрЃАрЃўрЃљрЃЌрЃћрЃЉрЃџрЃћрЃЉрЃў**:
  - **рЃЎрЃЮрЃЏрЃърЃљрЃЦрЃбрЃБрЃарЃў**: рЃДрЃЋрЃћрЃџрЃљ рЃАрЃљрЃГрЃўрЃарЃЮ рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃў рЃћрЃарЃЌ рЃцрЃљрЃўрЃџрЃерЃўрЃљ рЃерЃћрЃцрЃБрЃЌрЃБрЃџрЃў, рЃарЃљрЃф рЃљрЃЏрЃљрЃарЃбрЃўрЃЋрЃћрЃЉрЃА рЃњрЃљрЃќрЃўрЃљрЃарЃћрЃЉрЃљрЃА.
  - **рЃЊрЃљрЃцрЃљрЃарЃБрЃџрЃў рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃљ**: рЃљрЃа рЃљрЃарЃўрЃА рЃърЃўрЃарЃЊрЃљрЃърЃўрЃа рЃгрЃљрЃЎрЃўрЃЌрЃ«рЃЋрЃљрЃЊрЃў, рЃЏрЃљрЃњрЃарЃљрЃЏ QGIS рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃљрЃЊ рЃљрЃЏрЃБрЃерЃљрЃЋрЃћрЃЉрЃА рЃЏрЃљрЃА.
  - **рЃЏрЃЮрЃАрЃљрЃ«рЃћрЃарЃ«рЃћрЃЉрЃћрЃџрЃў рЃњрЃљрЃќрЃўрЃљрЃарЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА**: рЃўрЃЊрЃћрЃљрЃџрЃБрЃарЃўрЃљ рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃћрЃарЃЌрЃў рЃцрЃљрЃўрЃџрЃўрЃЌ рЃњрЃљрЃЊрЃљрЃфрЃћрЃЏрЃўрЃАрЃЌрЃЋрЃўрЃА.
- **рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃА рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃљ**:
  - рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃАрЃљрЃЉрЃЮрЃџрЃЮрЃЮ рЃњрЃљрЃЋрЃарЃфрЃћрЃџрЃћрЃЉрЃљ.
  - рЃАрЃљрЃарЃћрЃќрЃћрЃарЃЋрЃЮ рЃљрЃАрЃџрЃўрЃА (backup) рЃерЃћрЃЦрЃЏрЃюрЃљ.

­ЪЊї **рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў**:
```bash
my_project.qgz
```

## Рџќ№ИЈ рЃерЃћрЃЊрЃљрЃарЃћрЃЉрЃљ

| рЃЏрЃљрЃ«рЃљрЃАрЃўрЃљрЃЌрЃћрЃЉрЃћрЃџрЃў           | `.qgs`                | `.qgz`                       |
|------------------------|-----------------------|------------------------------|
| рЃцрЃЮрЃарЃЏрЃљрЃбрЃў                | рЃбрЃћрЃЦрЃАрЃбрЃБрЃарЃў (XML)        | рЃерЃћрЃЎрЃБрЃЏрЃерЃБрЃџрЃў (ZIP)              |
| рЃгрЃљрЃЎрЃўрЃЌрЃ«рЃЋрЃљрЃЊрЃЮрЃЉрЃљ           | рЃљрЃЊрЃЋрЃўрЃџрЃљрЃЊ рЃгрЃљрЃЎрЃўрЃЌрЃ«рЃЋрЃљрЃЊрЃў    | рЃљрЃа рЃљрЃарЃўрЃА рЃърЃўрЃарЃЊрЃљрЃърЃўрЃа рЃгрЃљрЃЎрЃўрЃЌрЃ«рЃЋрЃљрЃЊрЃў |
| Git рЃЏрЃ«рЃљрЃарЃЊрЃљрЃГрЃћрЃарЃљ         | Рюћ№ИЈ рЃерЃћрЃАрЃљрЃцрЃћрЃарЃўрЃАрЃўрЃљ         | РЮї рЃюрЃљрЃЎрЃџрЃћрЃЉрЃљрЃЊ рЃЏрЃЮрЃАрЃљрЃ«рЃћрЃарЃ«рЃћрЃЉрЃћрЃџрЃў     |
| рЃњрЃљрЃќрЃўрЃљрЃарЃћрЃЉрЃљ              | рЃАрЃљрЃГрЃўрЃарЃЮ рЃЊрЃљрЃЎрЃљрЃЋрЃерЃўрЃарЃћрЃЉрЃБрЃџрЃў рЃцрЃљрЃўрЃџрЃћрЃЉрЃўрЃф | рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃћрЃарЃЌрЃў рЃцрЃљрЃўрЃџрЃў       |
| рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃА рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃљ   | рЃњрЃљрЃюрЃЋрЃўрЃЌрЃљрЃарЃћрЃЉрЃљ, рЃЋрЃћрЃарЃАрЃўрЃћрЃЉрЃўрЃА рЃЎрЃЮрЃюрЃбрЃарЃЮрЃџрЃў | рЃАрЃљрЃЉрЃЮрЃџрЃЮрЃЮ рЃњрЃљрЃЋрЃарЃфрЃћрЃџрЃћрЃЉрЃљ, рЃАрЃљрЃарЃћрЃќрЃћрЃарЃЋрЃЮ рЃљрЃАрЃџрЃў |

## РюЁ рЃарЃћрЃЎрЃЮрЃЏрЃћрЃюрЃЊрЃљрЃфрЃўрЃћрЃЉрЃў

**рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћ `.qgs`, рЃЌрЃБ**:
- рЃљрЃЦрЃбрЃўрЃБрЃарЃљрЃЊ рЃЏрЃБрЃерЃљрЃЮрЃЉ рЃърЃарЃЮрЃћрЃЦрЃбрЃќрЃћ.
- рЃњрЃГрЃўрЃарЃЊрЃћрЃЉрЃљ рЃцрЃљрЃўрЃџрЃўрЃА рЃ«рЃћрЃџрЃўрЃЌ рЃарЃћрЃЊрЃљрЃЦрЃбрЃўрЃарЃћрЃЉрЃљ рЃљрЃю рЃЋрЃћрЃарЃАрЃўрЃћрЃЉрЃўрЃА рЃЎрЃЮрЃюрЃбрЃарЃЮрЃџрЃў (рЃЏрЃљрЃњ. Git-рЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃЌ).

**рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћ `.qgz`, рЃЌрЃБ**:
- рЃњрЃАрЃБрЃарЃА рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃерЃћрЃцрЃБрЃЌрЃЋрЃљ рЃЊрЃљ рЃњрЃљрЃќрЃўрЃљрЃарЃћрЃЉрЃљ рЃћрЃарЃЌрЃў рЃцрЃљрЃўрЃџрЃўрЃА рЃАрЃљрЃ«рЃўрЃЌ.
- рЃЦрЃЏрЃюрЃў рЃАрЃљрЃарЃћрЃќрЃћрЃарЃЋрЃЮ рЃљрЃАрЃџрЃА рЃљрЃю рЃљрЃњрЃќрЃљрЃЋрЃюрЃў рЃърЃарЃЮрЃћрЃЦрЃбрЃА рЃАрЃ«рЃЋрЃљ рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃўрЃАрЃЌрЃЋрЃўрЃА.

­ЪЊї **рЃарЃЕрЃћрЃЋрЃљ**: QGIS-рЃерЃў рЃерЃћрЃњрЃўрЃФрЃџрЃўрЃљ рЃЌрЃљрЃЋрЃўрЃАрЃБрЃцрЃџрЃљрЃЊ рЃњрЃљрЃЊрЃљрЃћрЃарЃЌрЃЮ рЃцрЃЮрЃарЃЏрЃљрЃбрЃћрЃЉрЃА рЃерЃЮрЃарЃўрЃА **File > Save As** рЃЮрЃцрЃфрЃўрЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃЌ,
