# ­ЪЊЂ QGIS РђЊ Save Vector Layer As: Format Options

QGIS-рЃўрЃА **`Save Vector Layer As...`** рЃцрЃљрЃюрЃ»рЃљрЃарЃљрЃерЃў рЃерЃћрЃњрЃўрЃФрЃџрЃўрЃљ рЃерЃћрЃюрЃљрЃ«рЃЮ рЃерЃарЃћрЃћрЃЉрЃў рЃАрЃ«рЃЋрЃљрЃЊрЃљрЃАрЃ«рЃЋрЃљ рЃцрЃЮрЃарЃЏрЃљрЃбрЃерЃў Рђћ рЃЌрЃўрЃЌрЃЮрЃћрЃБрЃџрЃў рЃЏрЃљрЃЌрЃњрЃљрЃюрЃў рЃЏрЃЮрЃарЃњрЃћрЃЉрЃБрЃџрЃўрЃљ рЃњрЃљрЃюрЃАрЃ«рЃЋрЃљрЃЋрЃћрЃЉрЃБрЃџ рЃАрЃљрЃГрЃўрЃарЃЮрЃћрЃЉрЃљрЃАрЃљ рЃЊрЃљ рЃњрЃљрЃарЃћрЃЏрЃЮрЃќрЃћ.

## РюЁ рЃарЃЕрЃћрЃЋрЃћрЃЉрЃў

- **GeoPackage (.gpkg)** Рђћ ­ЪЦЄ рЃарЃћрЃЎрЃЮрЃЏрЃћрЃюрЃЊрЃћрЃЉрЃБрЃџрЃў рЃЌрЃљрЃюрЃљрЃЏрЃћрЃЊрЃарЃЮрЃЋрЃћ рЃцрЃЮрЃарЃЏрЃљрЃбрЃў, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃЕрЃљрЃљрЃюрЃљрЃфрЃЋрЃџрЃћрЃЉрЃА Shapefile-рЃА.
- **GeoJSON / TopoJSON** Рђћ рЃАрЃљрЃБрЃЎрЃћрЃЌрЃћрЃАрЃЮрЃЊ рЃЏрЃБрЃерЃљрЃЮрЃЉрЃА Leaflet-рЃЌрЃљрЃю рЃЊрЃљ рЃЋрЃћрЃЉ рЃарЃБрЃЎрЃћрЃЉрЃЌрЃљрЃю.
- **Shapefile (.shp)** Рђћ рЃљрЃБрЃфрЃўрЃџрЃћрЃЉрЃћрЃџрЃўрЃљ рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃЌрЃБ рЃњрЃГрЃўрЃарЃЊрЃћрЃЉрЃљ рЃБрЃЎрЃБрЃњрЃљрЃбрЃљрЃюрЃљ рЃЏрЃЮрЃФрЃЋрЃћрЃџрЃћрЃЉрЃБрЃџ рЃАрЃўрЃАрЃбрЃћрЃЏрЃћрЃЉрЃЌрЃљрЃю.
- **DXF / DWG** Рђћ рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћ CAD-рЃќрЃћ рЃЏрЃБрЃерЃљрЃЮрЃЉрЃўрЃАрЃљрЃА.
- **PostgreSQL/PostGIS** Рђћ рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћ, рЃЌрЃБ рЃерЃћрЃюрЃў рЃЮрЃарЃњрЃљрЃюрЃўрЃќрЃљрЃфрЃўрЃљ рЃўрЃДрЃћрЃюрЃћрЃЉрЃА рЃАрЃўрЃЋрЃарЃфрЃўрЃЌ рЃЉрЃљрЃќрЃћрЃЉрЃА.

---

­Ъћё рЃцрЃЮрЃарЃЏрЃљрЃбрЃћрЃЉрЃўрЃА рЃЏрЃ«рЃљрЃарЃЊрЃљрЃГрЃћрЃарЃљ рЃЊрЃљрЃЏрЃЮрЃЎрЃўрЃЊрЃћрЃЉрЃБрЃџрЃўрЃљ GDAL-рЃўрЃА рЃЋрЃћрЃарЃАрЃўрЃљрЃАрЃљ рЃЊрЃљ QGIS-рЃўрЃА рЃљрЃњрЃћрЃЉрЃБрЃџрЃћрЃЉрЃљрЃќрЃћ.

­ЪЊї рЃўрЃ«рЃўрЃџрЃћ рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃўрЃЌрЃў [рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃљ:](https://gdal.org/drivers/vector/index.html)





QGIS-рЃерЃў **`Save Vector Layer As...`** рЃцрЃБрЃюрЃЦрЃфрЃўрЃљ рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃљрЃА рЃњрЃљрЃФрЃџрЃћрЃЋрЃЌ рЃерЃћрЃўрЃюрЃљрЃ«рЃЮрЃЌ рЃАрЃўрЃЋрЃарЃфрЃўрЃЌрЃў рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃў рЃЏрЃарЃљрЃЋрЃљрЃџ рЃцрЃЮрЃарЃЏрЃљрЃбрЃерЃў Рђћ рЃЌрЃўрЃЌрЃЮрЃћрЃБрЃџ рЃЏрЃљрЃЌрЃњрЃљрЃюрЃА рЃЌрЃљрЃЋрЃўрЃАрЃў рЃБрЃърЃўрЃарЃљрЃбрЃћрЃАрЃЮрЃЉрЃљ рЃЊрЃљ рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃА рЃАрЃцрЃћрЃарЃЮ рЃљрЃЦрЃЋрЃА.

## ­ЪЌ║№ИЈ ­ЮЌЕ­ЮЌ▓­ЮЌ░­ЮўЂ­ЮЌ╝­ЮЌ┐ ­ЮЌЎ­ЮЌ╝­ЮЌ┐­ЮЌ║­ЮЌ«­ЮўЂ­Юўђ (­ЮЌЋ­ЮЌ▓­Юўђ­ЮўЂ: **GeoPackage Рєњ `.gpkg`**)

### ­Ъћ╣ **GeoPackage (.gpkg)**
- РюЁ **рЃћрЃарЃЌ рЃцрЃљрЃўрЃџрЃерЃў** рЃўрЃюрЃљрЃ«рЃљрЃЋрЃА рЃЏрЃарЃљрЃЋрЃљрЃџ рЃерЃарЃарЃћрЃА (рЃгрЃћрЃарЃбрЃўрЃџрЃћрЃЉрЃў, рЃ«рЃљрЃќрЃћрЃЉрЃў, рЃърЃЮрЃџрЃўрЃњрЃЮрЃюрЃћрЃЉрЃў, рЃарЃљрЃАрЃбрЃћрЃарЃћрЃЉрЃў)
- РюЁ **UTF-8 рЃЏрЃ«рЃљрЃарЃЊрЃљрЃГрЃћрЃарЃљ** рЃбрЃћрЃЦрЃАрЃбрЃўрЃА рЃАрЃгрЃЮрЃарЃљрЃЊ рЃЎрЃЮрЃЊрЃўрЃарЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА
- РюЁ **рЃерЃћрЃюрЃљрЃ«рЃБрЃџрЃў рЃАрЃбрЃўрЃџрЃћрЃЉрЃў** рЃЊрЃљ рЃЏрЃћрЃбрЃљрЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃў
- ­ЪЪб рЃарЃћрЃЎрЃЮрЃЏрЃћрЃюрЃЊрЃўрЃарЃћрЃЉрЃБрЃџрЃўрЃљ рЃарЃЮрЃњрЃЮрЃарЃф **рЃЌрЃљрЃюрЃљрЃЏрЃћрЃЊрЃарЃЮрЃЋрЃћ, рЃАрЃбрЃљрЃЉрЃўрЃџрЃБрЃарЃў рЃЊрЃљ рЃЏрЃарЃљрЃЋрЃљрЃџрЃърЃџрЃљрЃбрЃцрЃЮрЃарЃЏрЃБрЃџрЃў** рЃцрЃљрЃўрЃџрЃўрЃА рЃцрЃЮрЃарЃЏрЃљрЃбрЃў

### ­ЪћИ **Shapefile (.shp)**
- ­ЪЊд рЃЎрЃџрЃљрЃАрЃўрЃЎрЃБрЃарЃў рЃцрЃЮрЃарЃЏрЃљрЃбрЃў, рЃФрЃљрЃџрЃўрЃљрЃю рЃцрЃљрЃарЃЌрЃЮрЃЊ рЃЏрЃ«рЃљрЃарЃЊрЃљрЃГрЃћрЃарЃўрЃџрЃў
- РЮЌ рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃћрЃарЃЌрЃў рЃерЃарЃћ рЃЌрЃўрЃЌрЃЮ рЃцрЃљрЃўрЃџрЃерЃў
- РЮЌ рЃцрЃўрЃџрЃўрЃА рЃАрЃљрЃ«рЃћрЃџрЃћрЃЉрЃў 10 рЃАрЃўрЃЏрЃЉрЃЮрЃџрЃЮрЃќрЃћ рЃерЃћрЃќрЃдрЃБрЃЊрЃБрЃџрЃўрЃљ
- РЮЌ рЃљрЃБрЃфрЃўрЃџрЃћрЃЉрЃћрЃџрЃўрЃљ рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃўрЃЌрЃў рЃцрЃљрЃўрЃџрЃћрЃЉрЃў (.shx, .dbf, .prj)

### ­Ъїљ **GeoJSON / KML**
- ­ЪїЇ рЃўрЃЊрЃћрЃљрЃџрЃБрЃарЃўрЃљ рЃЋрЃћрЃЉ рЃарЃБрЃЎрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА рЃЊрЃљ Google Earth-рЃЌрЃљрЃю рЃњрЃљрЃАрЃљрЃќрЃўрЃљрЃарЃћрЃЉрЃџрЃљрЃЊ
- РюЁ рЃбрЃћрЃЦрЃАрЃбрЃќрЃћ рЃЊрЃљрЃцрЃБрЃФрЃюрЃћрЃЉрЃБрЃџрЃў, рЃљрЃЊрЃљрЃЏрЃўрЃљрЃюрЃўрЃАрЃЌрЃЋрЃўрЃА рЃгрЃљрЃЎрЃўрЃЌрЃ«рЃЋрЃљрЃЊрЃў
- РюЁ рЃЏрЃљрЃарЃбрЃўрЃЋрЃљрЃЊ рЃЕрЃљрЃАрЃљрЃАрЃЏрЃћрЃџрЃўрЃљ JavaScript рЃЉрЃўрЃЉрЃџрЃўрЃЮрЃЌрЃћрЃЎрЃћрЃЉрЃерЃў (Leaflet, Mapbox)

### ­ЪЈЌ№ИЈ **DXF (.dxf)**
- ­Ъњ╝ CAD рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃА рЃњрЃљрЃфрЃЋрЃџрЃўрЃА рЃцрЃЮрЃарЃЏрЃљрЃбрЃў
- ­Ъј» рЃќрЃБрЃАрЃбрЃў рЃњрЃћрЃЮрЃЏрЃћрЃбрЃарЃўрЃљ AutoCAD рЃЊрЃљ рЃАрЃ«рЃЋрЃљ CAD рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃћрЃЉрЃЌрЃљрЃю рЃўрЃюрЃбрЃћрЃњрЃарЃљрЃфрЃўрЃўрЃАрЃЌрЃЋрЃўрЃА

---

## ­ЪЪе ­ЮЌЦ­ЮЌ«­Юўђ­ЮўЂ­ЮЌ▓­ЮЌ┐ ­ЮЌЎ­ЮЌ╝­ЮЌ┐­ЮЌ║­ЮЌ«­ЮўЂ­Юўђ (­ЮЌЋ­ЮЌ▓­Юўђ­ЮўЂ: **GeoTIFF Рєњ `.tif`**)

### ­Ъќ╝№ИЈ **GeoTIFF (.tif)**
- ­ЪЏ░№ИЈ рЃАрЃбрЃљрЃюрЃЊрЃљрЃарЃбрЃў рЃАрЃљрЃбрЃћрЃџрЃўрЃбрЃБрЃарЃў рЃњрЃљрЃЏрЃЮрЃАрЃљрЃ«рЃБрЃџрЃћрЃЉрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА, DEM-рЃћрЃЉрЃўрЃАрЃљ рЃЊрЃљ рЃўрЃюрЃЊрЃћрЃЦрЃАрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА (NDVI, hillshade)
- РюЁ рЃЏрЃарЃљрЃЋрЃљрЃџрЃЉрЃљрЃюрЃЊрЃўрЃљрЃюрЃў рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃА рЃЊрЃљ рЃЏрЃћрЃбрЃљрЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃА рЃЏрЃ«рЃљрЃарЃЊрЃљрЃГрЃћрЃарЃљ
- РюЁ рЃцрЃљрЃарЃЌрЃЮрЃЊ рЃЏрЃ«рЃљрЃарЃЊрЃљрЃГрЃћрЃарЃўрЃџрЃў рЃърЃџрЃљрЃбрЃцрЃЮрЃарЃЏрЃћрЃЉрЃќрЃћ

### ­ЪДЙ **IMG / ASCII**
- ­ЪЋ░№ИЈ рЃЏрЃћрЃЏрЃЎрЃЋрЃўрЃЊрЃарЃћрЃЮрЃЉрЃўрЃЌрЃў рЃЊрЃљ рЃАрЃљрЃЏрЃЮрЃЊрЃћрЃџрЃЮ рЃцрЃЮрЃарЃЏрЃљрЃбрЃћрЃЉрЃў
- ­ЪњА рЃАрЃљрЃАрЃљрЃарЃњрЃћрЃЉрЃџрЃЮ рЃњрЃљрЃарЃћрЃЏрЃЮрЃА рЃАрЃўрЃЏрЃБрЃџрЃљрЃфрЃўрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА рЃљрЃю рЃАрЃърЃћрЃфрЃўрЃљрЃџрЃўрЃќрЃћрЃЉрЃБрЃџрЃў GIS рЃЏрЃЮрЃЊрЃБрЃџрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА

---

## ­ЪЊі ­ЮЌД­ЮЌ«­ЮЌ»­Юўѓ­ЮЌ╣­ЮЌ«­ЮЌ┐ ­ЮЌЎ­ЮЌ╝­ЮЌ┐­ЮЌ║­ЮЌ«­ЮўЂ­Юўђ (­ЮЌЋ­ЮЌ▓­Юўђ­ЮўЂ: **CSV Рєњ `.csv`**)

### ­ЪЊё **CSV / Excel (.csv / .xlsx)**
- ­ЪћЌ рЃЏрЃљрЃарЃбрЃўрЃЋрЃў рЃњрЃќрЃљрЃљ рЃљрЃбрЃарЃўрЃЉрЃБрЃбрЃБрЃџрЃў рЃфрЃ«рЃарЃўрЃџрЃћрЃЉрЃўрЃА рЃўрЃЏрЃърЃЮрЃарЃбрЃўрЃАрЃљ рЃЊрЃљ рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃА рЃерЃарЃћрЃћрЃЉрЃќрЃћ рЃЏрЃўрЃЉрЃЏрЃўрЃАрЃЌрЃЋрЃўрЃА
- ­ЪДГ рЃЌрЃБ рЃњрЃљрЃЦрЃЋрЃЌ рЃЎрЃЮрЃЮрЃарЃЊрЃўрЃюрЃљрЃбрЃћрЃЉрЃў, рЃерЃћрЃњрЃўрЃФрЃџрЃўрЃљрЃЌ рЃърЃўрЃарЃЊрЃљрЃърЃўрЃа рЃгрЃћрЃарЃбрЃўрЃџрЃћрЃЉрЃўрЃА рЃарЃБрЃЎрЃљрЃЊ рЃњрЃљрЃарЃЊрЃљрЃЦрЃЏрЃюрЃљ

---

## РюЁ ­ЮЌъ­ЮЌ▓­Юўє ­ЮЌД­ЮЌ«­ЮЌИ­ЮЌ▓­ЮЌ«­Юўё­ЮЌ«­Юўє­Юўђ

- ­Ъј» рЃАрЃгрЃЮрЃарЃў рЃцрЃЮрЃарЃЏрЃљрЃбрЃўрЃА рЃерЃћрЃарЃЕрЃћрЃЋрЃљ рЃБрЃќрЃарЃБрЃюрЃЋрЃћрЃџрЃДрЃЮрЃцрЃА:
  - **рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃЏрЃЌрЃџрЃўрЃљрЃюрЃЮрЃЉрЃљрЃА**
  - **рЃърЃџрЃљрЃбрЃцрЃЮрЃарЃЏрЃљрЃЌрЃерЃЮрЃарЃўрЃА рЃЌрЃљрЃЋрЃАрЃћрЃЉрЃљрЃЊрЃЮрЃЉрЃљрЃА** (ArcGIS, GRASS, FME)
  - **рЃЌрЃљрЃюрЃљрЃЏрЃерЃарЃЮрЃЏрЃџрЃЮрЃЉрЃўрЃА рЃњрЃљрЃЏрЃљрЃарЃбрЃўрЃЋрЃћрЃЉрЃљрЃА** рЃърЃарЃЮрЃћрЃЦрЃбрЃћрЃЉрЃерЃў
- ­ЪЈЎ№ИЈ рЃўрЃЦрЃюрЃћрЃЉрЃљ рЃћрЃА рЃБрЃарЃЉрЃљрЃюрЃБрЃџрЃў рЃЊрЃљрЃњрЃћрЃњрЃЏрЃљрЃарЃћрЃЉрЃљ, рЃўрЃюрЃцрЃарЃљрЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃўрЃА рЃЊрЃўрЃќрЃљрЃўрЃюрЃў рЃЌрЃБ рЃАрЃљрЃЎрЃљрЃЊрЃљрЃАрЃбрЃарЃЮ рЃЉрЃљрЃќрЃћрЃЉрЃў Рђћ рЃцрЃЮрЃарЃЏрЃљрЃбрЃў рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЋрЃљрЃюрЃўрЃљ.
