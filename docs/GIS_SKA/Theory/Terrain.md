# ­ЪїЇ рЃарЃћрЃџрЃўрЃћрЃцрЃБрЃарЃў рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃў рЃЊрЃљ QGIS-рЃерЃў рЃЏрЃљрЃЌрЃў рЃЊрЃљрЃЏрЃБрЃерЃљрЃЋрЃћрЃЉрЃљ

## ­ЪћЇ рЃарЃћрЃџрЃўрЃћрЃцрЃБрЃарЃў рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃА рЃљрЃарЃАрЃў

рЃарЃћрЃџрЃўрЃћрЃцрЃў рЃгрЃљрЃарЃЏрЃЮрЃљрЃЊрЃњрЃћрЃюрЃА рЃЊрЃћрЃЊрЃљрЃЏрЃўрЃгрЃўрЃА рЃќрЃћрЃЊрЃљрЃърЃўрЃарЃўрЃА рЃАрЃўрЃЏрЃљрЃдрЃџрЃћрЃћрЃЉрЃўрЃА, рЃЊрЃљрЃЦрЃљрЃюрЃћрЃЉрЃћрЃЉрЃўрЃАрЃљ рЃЊрЃљ рЃцрЃЮрЃарЃЏрЃћрЃЉрЃўрЃА рЃћрЃарЃЌрЃЮрЃЉрЃџрЃўрЃЮрЃЉрЃљрЃА. рЃЏрЃўрЃАрЃў рЃљрЃюрЃљрЃџрЃўрЃќрЃў рЃљрЃБрЃфрЃўрЃџрЃћрЃЉрЃћрЃџрЃўрЃљ:

- рЃњрЃћрЃЮрЃЏрЃЮрЃарЃцрЃЮрЃџрЃЮрЃњрЃўрЃБрЃарЃў рЃЎрЃЋрЃџрЃћрЃЋрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА  
- рЃАрЃљрЃцрЃарЃЌрЃ«рЃћрЃћрЃЉрЃўрЃА рЃерЃћрЃцрЃљрЃАрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА (рЃЏрЃћрЃгрЃДрЃћрЃарЃў, рЃгрЃДрЃљрЃџрЃЊрЃўрЃЊрЃЮрЃЉрЃљ рЃЊрЃљ рЃАрЃ«рЃЋ.)  
- рЃўрЃюрЃцрЃарЃљрЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃўрЃА рЃЊрЃљрЃњрЃћрЃњрЃЏрЃЋрЃўрЃАрЃЌрЃЋрЃўрЃА  
- рЃџрЃљрЃюрЃЊрЃерЃљрЃцрЃбрЃБрЃарЃў рЃЏрЃЮрЃЊрЃћрЃџрЃўрЃарЃћрЃЉрЃўрЃАрЃљ рЃЊрЃљ рЃЋрЃўрЃќрЃБрЃљрЃџрЃўрЃќрЃљрЃфрЃўрЃўрЃАрЃЌрЃЋрЃўрЃА  

---

## ­ЪЏ░№ИЈ рЃарЃћрЃџрЃўрЃћрЃцрЃБрЃарЃў рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃА рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃў рЃгрЃДрЃљрЃарЃЮрЃћрЃЉрЃў

### 1. **SRTM (Shuttle Radar Topography Mission)**
- рЃњрЃљрЃцрЃљрЃарЃЌрЃЮрЃћрЃЉрЃљ: `.tif` (GeoTIFF)
- рЃњрЃџрЃЮрЃЉрЃљрЃџрЃБрЃарЃў рЃЊрЃљрЃцрЃљрЃарЃЋрЃљ: 30рЃЏ рЃЊрЃљ 90рЃЏ рЃарЃћрЃќрЃЮрЃџрЃБрЃфрЃўрЃўрЃЌ
- рЃгрЃДрЃљрЃарЃЮ: https://earthexplorer.usgs.gov/

### 2. **ASTER GDEM**
- рЃњрЃљрЃцрЃљрЃарЃЌрЃЮрЃћрЃЉрЃљ: `.tif`
- рЃњрЃџрЃЮрЃЉрЃљрЃџрЃБрЃарЃў рЃЊрЃљрЃцрЃљрЃарЃЋрЃљ ~30рЃЏ рЃарЃћрЃќрЃЮрЃџрЃБрЃфрЃўрЃўрЃЌ
- рЃгрЃДрЃљрЃарЃЮ: https://asterweb.jpl.nasa.gov/gdem.asp

### 3. **Copernicus DEM**
- рЃњрЃљрЃцрЃљрЃарЃЌрЃЮрЃћрЃЉрЃљ: `.tif`
- рЃЏрЃљрЃдрЃљрЃџрЃў рЃАрЃўрЃќрЃБрЃАрЃбрЃўрЃА рЃарЃћрЃџрЃўрЃћрЃцрЃў (30рЃЏ рЃЊрЃљ 10рЃЏ)
- рЃгрЃДрЃљрЃарЃЮ: https://dataspace.copernicus.eu/

### 4. **ALOS PALSAR**
- рЃњрЃљрЃцрЃљрЃарЃЌрЃЮрЃћрЃЉрЃљ: `.tif`
- рЃЊрЃћрЃбрЃљрЃџрЃБрЃарЃў рЃарЃћрЃџрЃўрЃћрЃцрЃБрЃарЃў рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃў (12.5рЃЏ)
- рЃгрЃДрЃљрЃарЃЮ: https://asf.alaska.edu/

### 5. **GeoPortal.ge (NAPR)**
- рЃАрЃљрЃЦрЃљрЃарЃЌрЃЋрЃћрЃџрЃЮрЃА рЃћрЃарЃЮрЃЋрЃюрЃБрЃџрЃў рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃў
- рЃгрЃДрЃљрЃарЃЮ: https://www.geoportal.ge/

## ­Ъњ╗ QGIS-рЃерЃў рЃарЃћрЃџрЃўрЃћрЃцрЃБрЃарЃў рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃА рЃЊрЃљрЃЏрЃБрЃерЃљрЃЋрЃћрЃЉрЃљ

### ­ЪЊЦ рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃА рЃЕрЃљрЃбрЃЋрЃўрЃарЃЌрЃЋрЃљ

1. рЃњрЃљрЃ«рЃАрЃћрЃюрЃўрЃЌ QGIS  
2. Layer > Add Layer > Add Raster Layer  
3. рЃљрЃўрЃарЃЕрЃўрЃћрЃЌ рЃЌрЃЦрЃЋрЃћрЃюрЃў `.tif` рЃцрЃЮрЃарЃЏрЃљрЃбрЃўрЃА рЃарЃћрЃџрЃўрЃћрЃцрЃБрЃарЃў рЃцрЃљрЃўрЃџрЃў  

### ­ЪЌ║№ИЈ рЃцрЃћрЃарЃљрЃЊрЃў рЃарЃћрЃџрЃўрЃћрЃцрЃўрЃА рЃЋрЃўрЃќрЃБрЃљрЃџрЃўрЃќрЃљрЃфрЃўрЃљ

- Layer Styling Panel > Render type: Singleband pseudocolor  
- рЃљрЃўрЃарЃЕрЃўрЃћрЃЌ Color ramp (рЃЏрЃљрЃњ: `terrain`, `viridis`)  
- рЃњрЃљрЃљрЃљрЃЦрЃбрЃўрЃБрЃарЃћрЃЌ рЃЎрЃџрЃљрЃАрЃў (`Mode`: Equal interval, Quantile рЃЊрЃљ рЃАрЃ«рЃЋ.)

### ­ЪЊѕ рЃЊрЃљрЃ«рЃарЃўрЃА (Slope), рЃћрЃЦрЃАрЃърЃЮрЃќрЃўрЃфрЃўрЃўрЃА (Aspect) рЃЊрЃљ рЃЕрЃарЃЊрЃўрЃџрЃћрЃЉрЃўрЃА (Hillshade) рЃњрЃћрЃюрЃћрЃарЃўрЃарЃћрЃЉрЃљ

1. Processing Toolbox > Raster Terrain Analysis  
2. рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ:
   - **Slope** РђЊ рЃЊрЃљрЃ«рЃарЃўрЃџрЃЮрЃЉрЃўрЃА рЃарЃБрЃЎрЃљ  
   - **Aspect** РђЊ рЃ░рЃЮрЃарЃўрЃќрЃЮрЃюрЃбрЃўрЃА рЃЏрЃ«рЃљрЃарЃћ   
   - **Hillshade** РђЊ рЃАрЃўрЃюрЃљрЃЌрЃџрЃўрЃА рЃўрЃЏрЃўрЃбрЃљрЃфрЃўрЃљ рЃарЃћрЃџрЃўрЃћрЃцрЃќрЃћ  

### ­Ъїђ рЃўрЃќрЃЮрЃ«рЃљрЃќрЃћрЃЉрЃўрЃА (Contours) рЃњрЃћрЃюрЃћрЃарЃўрЃарЃћрЃЉрЃљ

1. Processing Toolbox > Contour  
2. рЃљрЃўрЃарЃЕрЃўрЃћрЃЌ DEM  
3. Example:
   - Interval: 10 рЃЏ
   - Attribute name: ELEV  

### ­ЪД« рЃарЃћрЃЎрЃџрЃљрЃАрЃўрЃцрЃўрЃЎрЃљрЃфрЃўрЃљ (Reclassification)

1. Processing Toolbox > `Reclassify by table`  
2. рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЌ рЃЎрЃџрЃљрЃАрЃў рЃАрЃўрЃЏрЃљрЃдрЃџрЃўрЃА рЃЊрЃўрЃљрЃърЃљрЃќрЃЮрЃюрЃћрЃЉрЃўрЃЌ  
   - рЃЏрЃљрЃњ: 0РђЊ200 = рЃЊрЃљрЃЉрЃџрЃЮрЃЉрЃў  
   - 200РђЊ600 = рЃЉрЃЮрЃарЃфрЃЋрЃћрЃЉрЃў  
   - 600+ = рЃЏрЃЌрЃћрЃЉрЃў  

### ­Ъїљ TIN рЃЊрЃљ 3D рЃЋрЃўрЃќрЃБрЃљрЃџрЃўрЃќрЃљрЃфрЃўрЃљ

- рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЌ `TIN interpolation` рЃЊрЃљ `3D Map View`  
- рЃЏрЃљрЃАрЃерЃбрЃљрЃЉрЃБрЃарЃў рЃарЃћрЃџрЃўрЃћрЃцрЃўрЃА рЃгрЃљрЃарЃЏрЃЮрЃЊрЃњрЃћрЃюрЃљ рЃЏрЃЮрЃфрЃБрЃџрЃЮрЃЉрЃўрЃЌрЃў рЃцрЃЮрЃарЃЏрЃўрЃЌ
