# ­Ъїљ рЃАрЃўрЃЋрЃарЃфрЃБрЃџрЃў рЃЋрЃћрЃЉрЃАрЃћрЃарЃЋрЃўрЃАрЃћрЃЉрЃў: WMS, WFS, WMTS, WCS рЃЊрЃљ рЃАрЃ«рЃЋрЃљ

рЃАрЃўрЃЋрЃарЃфрЃБрЃџрЃў рЃЋрЃћрЃЉрЃАрЃћрЃарЃЋрЃўрЃАрЃћрЃЉрЃў рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃљрЃА рЃњрЃЋрЃљрЃФрЃџрЃћрЃЋрЃА рЃњрЃћрЃЮрЃњрЃарЃљрЃцрЃўрЃБрЃџрЃў рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃА рЃњрЃљрЃфрЃЋрЃџрЃљ, рЃЕрЃЋрЃћрЃюрЃћрЃЉрЃљ рЃЊрЃљ рЃљрЃюрЃљрЃџрЃўрЃќрЃў рЃўрЃюрЃбрЃћрЃарЃюрЃћрЃбрЃўрЃЌ. рЃўрЃАрЃўрЃюрЃў рЃЊрЃљрЃцрЃБрЃФрЃюрЃћрЃЉрЃБрЃџрЃўрЃљ **OGC (Open Geospatial Consortium)** рЃАрЃбрЃљрЃюрЃЊрЃљрЃарЃбрЃћрЃЉрЃќрЃћ.

---

## ­ЪЌ║№ИЈ WMS РђЊ Web Map Service

- **рЃарЃљрЃА рЃљрЃЎрЃћрЃЌрЃћрЃЉрЃА:** рЃљрЃњрЃћрЃюрЃћрЃарЃўрЃарЃћрЃЉрЃА рЃарЃБрЃЎрЃўрЃА рЃњрЃљрЃЏрЃЮрЃАрЃљрЃ«рЃБрЃџрЃћрЃЉрЃљрЃА (PNG, JPEG) рЃАрЃћрЃарЃЋрЃћрЃарЃќрЃћ рЃЊрЃљ рЃљрЃњрЃќрЃљрЃЋрЃюрЃўрЃА рЃЎрЃџрЃўрЃћрЃюрЃбрЃА.
- **рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ:** рЃарЃБрЃЎрЃўрЃА рЃЋрЃўрЃќрЃБрЃљрЃџрЃўрЃќрЃљрЃфрЃўрЃљ РђЊ рЃЏрЃљрЃњрЃарЃљрЃЏ рЃљрЃарЃљ рЃљрЃбрЃарЃўрЃЉрЃБрЃбрЃБрЃџрЃў рЃљрЃюрЃљрЃџрЃўрЃќрЃў.
- **рЃцрЃЮрЃарЃЏрЃљрЃбрЃў:** рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃњрЃљрЃЏрЃЮрЃАрЃљрЃ«рЃБрЃџрЃћрЃЉрЃљ (рЃАрЃБрЃарЃљрЃЌрЃў), рЃЋрЃћрЃа рЃгрЃљрЃўрЃЎрЃўрЃЌрЃ«рЃљрЃЋ рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃА рЃБрЃерЃБрЃљрЃџрЃЮрЃЊ.

**рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў рЃЏрЃЮрЃЌрЃ«рЃЮрЃЋрЃюрЃљ:**  
```
https://myserver/geoserver/wms?SERVICE=WMS&REQUEST=GetMap&...
```

­ЪЊї рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ Leaflet, OpenLayers, QGIS рЃЊрЃљ рЃАрЃ«рЃЋрЃљ рЃЋрЃўрЃБрЃћрЃарЃћрЃЉрЃерЃў.

---

## ­ЪДЙ WFS РђЊ Web Feature Service

- **рЃарЃљрЃА рЃљрЃЎрЃћрЃЌрЃћрЃЉрЃА:** рЃњрЃќрЃљрЃЋрЃюрЃўрЃА рЃњрЃћрЃЮрЃњрЃарЃљрЃцрЃўрЃБрЃџ рЃЮрЃЉрЃўрЃћрЃЦрЃбрЃћрЃЉрЃА (рЃЏрЃљрЃњ. рЃерЃћрЃюрЃЮрЃЉрЃћрЃЉрЃў, рЃњрЃќрЃћрЃЉрЃў) рЃљрЃбрЃарЃўрЃЉрЃБрЃбрЃћрЃЉрЃўрЃЌ.
- **рЃцрЃЮрЃарЃЏрЃљрЃбрЃў:** GML, GeoJSON, рЃљрЃю рЃАрЃ«рЃЋрЃљ рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃќрЃћ рЃЮрЃарЃўрЃћрЃюрЃбрЃўрЃарЃћрЃЉрЃБрЃџрЃў рЃцрЃЮрЃарЃЏрЃљрЃбрЃћрЃЉрЃў.
- **рЃАрЃљрЃарЃњрЃћрЃЉрЃћрЃџрЃў:** рЃўрЃФрЃџрЃћрЃЋрЃљ querying-рЃА, рЃцрЃўрЃџрЃбрЃарЃљрЃфрЃўрЃљрЃА рЃЊрЃљ рЃфрЃЋрЃџрЃљрЃЊ рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљрЃю рЃЏрЃБрЃерЃљрЃЮрЃЉрЃљрЃА.

**рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў рЃЏрЃЮрЃЌрЃ«рЃЮрЃЋрЃюрЃљ:**  
```
https://myserver/geoserver/wfs?SERVICE=WFS&REQUEST=GetFeature&...
```

­ЪЊї рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ рЃљрЃюрЃљрЃџрЃўрЃќрЃўрЃАрЃЌрЃЋрЃўрЃА, рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃћрЃЦрЃАрЃърЃЮрЃарЃбрЃўрЃАрЃЌрЃЋрЃўрЃА, рЃцрЃЮрЃарЃЏрЃћрЃЉрЃўрЃА рЃерЃћрЃЋрЃАрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА.

---

## ­ЪЌ║№ИЈ WMTS РђЊ Web Map Tile Service

- **рЃарЃљрЃА рЃљрЃЎрЃћрЃЌрЃћрЃЉрЃА:** рЃњрЃќрЃљрЃЋрЃюрЃўрЃА рЃгрЃўрЃюрЃљрЃАрЃгрЃљрЃа рЃЊрЃљрЃњрЃћрЃюрЃћрЃарЃўрЃарЃћрЃЉрЃБрЃџ рЃарЃБрЃЎрЃўрЃА рЃцрЃўрЃџрЃћрЃЉрЃА (tiles) рЃЏрЃГрЃўрЃЊрЃарЃЮ рЃЏрЃљрЃАрЃерЃбрЃљрЃЉрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА.
- **рЃАрЃўрЃАрЃгрЃарЃљрЃцрЃћ:** рЃБрЃцрЃарЃЮ рЃАрЃгрЃарЃљрЃцрЃўрЃљ рЃЋрЃўрЃЊрЃарЃћ WMS, рЃарЃљрЃЊрЃњрЃљрЃю рЃЎрЃџрЃљрЃўрЃћрЃюрЃбрЃА рЃљрЃњрЃќрЃљрЃЋрЃюрЃўрЃА рЃБрЃЎрЃЋрЃћ рЃњрЃљрЃДрЃЮрЃцрЃўрЃџ рЃАрЃБрЃарЃљрЃЌрЃћрЃЉрЃА.
- **рЃцрЃЮрЃарЃЏрЃљрЃбрЃў:** PNG, JPEG рЃцрЃўрЃџрЃћрЃЉрЃў.

**рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў URL (RESTful):**  
```
https://myserver/geoserver/gwc/service/wmts?layer=mylayer&...
```

­ЪЊї рЃўрЃЊрЃћрЃљрЃџрЃБрЃарЃўрЃљ рЃЏрЃљрЃдрЃљрЃџрЃў рЃърЃарЃЮрЃЊрЃБрЃЦрЃбрЃўрЃБрЃџрЃЮрЃЉрЃўрЃА рЃарЃБрЃЎрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА (рЃЏрЃљрЃњ. рЃцрЃЮрЃюрЃБрЃарЃў рЃарЃБрЃЎрЃћрЃЉрЃў).

---

## ­ЪДі WCS РђЊ Web Coverage Service

- **рЃарЃљрЃА рЃљрЃЎрЃћрЃЌрЃћрЃЉрЃА:** рЃњрЃљрЃЊрЃљрЃАрЃфрЃћрЃЏрЃА рЃњрЃћрЃЮрЃњрЃарЃљрЃцрЃўрЃБрЃџ *рЃарЃљрЃАрЃбрЃарЃБрЃџ рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃА* (coverage), рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃљрЃЊ рЃарЃћрЃџрЃўрЃћрЃцрЃў рЃљрЃю рЃАрЃљрЃбрЃћрЃџрЃўрЃбрЃБрЃарЃў рЃАрЃБрЃарЃљрЃЌрЃћрЃЉрЃў.
- **рЃцрЃЮрЃарЃЏрЃљрЃбрЃў:** TIFF, GRID, NetCDF рЃЊрЃљ рЃАрЃ«рЃЋрЃљ.
- **рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ:** рЃљрЃюрЃљрЃџрЃўрЃќрЃўрЃАрЃЌрЃЋрЃўрЃА, рЃарЃљрЃАрЃбрЃарЃБрЃџрЃў рЃЏрЃЮрЃЊрЃћрЃџрЃўрЃарЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА.

**рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў рЃЏрЃЮрЃЌрЃ«рЃЮрЃЋрЃюрЃљ:**  
```
https://myserver/geoserver/wcs?SERVICE=WCS&REQUEST=GetCoverage&...
```

­ЪЊї рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ рЃњрЃљрЃарЃћрЃЏрЃЮрЃА, рЃЎрЃџрЃўрЃЏрЃљрЃбрЃўрЃА, рЃЏрЃўрЃгрЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃА рЃЏрЃЮрЃЊрЃћрЃџрЃўрЃарЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА.

---

## ­ЪћЌ рЃАрЃ«рЃЋрЃљ рЃАрЃћрЃарЃЋрЃўрЃАрЃћрЃЉрЃў

### ­ЪЏ░№ИЈ CSW РђЊ Catalogue Service for the Web
- рЃЏрЃћрЃбрЃљрЃЊрЃЮрЃюрЃћрЃћрЃЉрЃўрЃА рЃФрЃћрЃЉрЃюрЃљ рЃЊрЃљ рЃЏрЃўрЃгрЃЮрЃЊрЃћрЃЉрЃљ.
- рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ рЃАрЃўрЃЋрЃарЃфрЃБрЃџрЃў рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃўрЃюрЃцрЃарЃљрЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃћрЃЉрЃерЃў (SDI).

### ­ЪЊА SOS РђЊ Sensor Observation Service
- рЃАрЃћрЃюрЃАрЃЮрЃарЃБрЃџрЃў рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃА рЃгрЃЋрЃЊрЃЮрЃЏрЃљ (рЃЏрЃљрЃњ: рЃбрЃћрЃЏрЃърЃћрЃарЃљрЃбрЃБрЃарЃљ, рЃгрЃюрЃћрЃЋрЃљ).
- рЃўрЃюрЃбрЃћрЃарЃюрЃћрЃб-рЃАрЃћрЃюрЃАрЃЮрЃарЃћрЃЉрЃўрЃА рЃЏрЃ«рЃљрЃарЃЊрЃљрЃГрЃћрЃарЃљ.

### ­ЪДф WPS РђЊ Web Processing Service
- рЃАрЃўрЃЋрЃарЃфрЃБрЃџрЃў рЃљрЃюрЃљрЃџрЃўрЃќрЃўрЃА рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃћрЃЉрЃўрЃА рЃњрЃљрЃерЃЋрЃћрЃЉрЃљ рЃўрЃюрЃбрЃћрЃарЃюрЃћрЃбрЃўрЃЌ (buffer, clip, intersection).
- рЃўрЃДрЃћрЃюрЃћрЃЉрЃА рЃЎрЃЮрЃЏрЃърЃџрЃћрЃЦрЃАрЃБрЃа рЃАрЃћрЃарЃЋрЃћрЃарЃБрЃџ рЃърЃарЃЮрЃфрЃћрЃАрЃўрЃюрЃњрЃА.

---

## ­ЪЊџ рЃерЃћрЃЊрЃљрЃарЃћрЃЉрЃўрЃА рЃфрЃ«рЃарЃўрЃџрЃў

| рЃАрЃћрЃарЃЋрЃўрЃАрЃў | рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃўрЃА рЃбрЃўрЃърЃў | рЃцрЃЮрЃарЃЏрЃљрЃбрЃў | рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ |
|---------|----------------|---------|------------|
| WMS     | рЃњрЃљрЃЏрЃЮрЃАрЃљрЃ«рЃБрЃџрЃћрЃЉрЃљ    | PNG, JPEG | рЃарЃБрЃЎрЃўрЃА рЃЋрЃўрЃќрЃБрЃљрЃџрЃўрЃќрЃљрЃфрЃўрЃљ |
| WFS     | рЃЮрЃЉрЃўрЃћрЃЦрЃбрЃћрЃЉрЃў       | GML, GeoJSON | рЃљрЃбрЃарЃўрЃЉрЃБрЃбрЃБрЃџрЃў рЃљрЃюрЃљрЃџрЃўрЃќрЃў |
| WMTS    | рЃцрЃўрЃџрЃћрЃЉрЃў          | PNG, JPEG | рЃАрЃгрЃарЃљрЃцрЃў рЃбрЃљрЃўрЃџрЃћрЃЉрЃў |
| WCS     | рЃарЃљрЃАрЃбрЃарЃћрЃЉрЃў        | GeoTIFF, GRID | рЃљрЃюрЃљрЃџрЃўрЃбрЃўрЃЎрЃБрЃарЃў рЃарЃљрЃАрЃбрЃарЃћрЃЉрЃў |
| CSW     | рЃЏрЃћрЃбрЃљрЃЊрЃЮрЃюрЃћ        | XML         | рЃарЃћрЃАрЃБрЃарЃАрЃћрЃЉрЃўрЃА рЃФрЃћрЃЉрЃюрЃљ |
| SOS     | рЃАрЃћрЃюрЃАрЃЮрЃарЃћрЃЉрЃў       | XML, JSON   | рЃАрЃћрЃюрЃАрЃЮрЃарЃўрЃА рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃў |
| WPS     | рЃърЃарЃЮрЃфрЃћрЃАрЃћрЃЉрЃў       | XML         | рЃАрЃўрЃЋрЃарЃфрЃБрЃџрЃў рЃљрЃюрЃљрЃџрЃўрЃќрЃў |

## ­Ъј» рЃЊрЃљрЃАрЃЎрЃЋрЃюрЃљ 

OGC-рЃўрЃА рЃЋрЃћрЃЉрЃАрЃћрЃарЃЋрЃўрЃАрЃћрЃЉрЃў рЃБрЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЋрЃљрЃюрЃћрЃАрЃўрЃљ рЃюрЃћрЃЉрЃўрЃАрЃЏрЃўрЃћрЃарЃў рЃЋрЃћрЃЉGIS рЃАрЃўрЃАрЃбрЃћрЃЏрЃўрЃАрЃЌрЃЋрЃўрЃА. рЃЌрЃўрЃЌрЃЮрЃћрЃБрЃџрЃў рЃАрЃћрЃарЃЋрЃўрЃАрЃў рЃћрЃЏрЃАрЃљрЃ«рЃБрЃарЃћрЃЉрЃљ рЃЎрЃЮрЃюрЃЎрЃарЃћрЃбрЃБрЃџ рЃЏрЃўрЃќрЃљрЃюрЃА РђЊ рЃњрЃљрЃЏрЃЮрЃАрЃљрЃ«рЃБрЃџрЃћрЃЉрЃљ, рЃљрЃбрЃарЃўрЃЉрЃБрЃбрЃБрЃџрЃў рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃў, рЃарЃљрЃАрЃбрЃарЃћрЃЉрЃў рЃљрЃю рЃљрЃюрЃљрЃџрЃўрЃќрЃў. рЃАрЃгрЃЮрЃарЃў рЃАрЃћрЃарЃЋрЃўрЃАрЃўрЃА рЃерЃћрЃарЃЕрЃћрЃЋрЃљ рЃЊрЃљрЃЏрЃЮрЃЎрЃўрЃЊрЃћрЃЉрЃБрЃџрЃўрЃљ рЃўрЃЏрЃљрЃќрЃћ, **рЃарЃљ рЃњрЃАрЃБрЃарЃЌ РђЊ рЃюрЃљрЃ«рЃЋрЃљ, рЃЎрЃўрЃЌрЃ«рЃЋрЃЋрЃљ, рЃљрЃюрЃљрЃџрЃўрЃќрЃў рЃЌрЃБ рЃцрЃўрЃџрЃбрЃарЃљрЃфрЃўрЃљ**.
