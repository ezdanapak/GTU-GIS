# ­Ъњ╗ Windows Environment рЃЊрЃљ GeoServer-рЃўрЃА рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃљ

---

## ­ЪДГ рЃарЃљ рЃљрЃарЃўрЃА **Windows Environment**?

**Windows Environment** рЃюрЃўрЃерЃюрЃљрЃЋрЃА рЃўрЃЏ рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃБрЃџ рЃАрЃўрЃАрЃбрЃћрЃЏрЃљрЃА, рЃарЃЮрЃЏрЃћрЃџрЃерЃўрЃф рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃћрЃџрЃў рЃЏрЃБрЃерЃљрЃЮрЃЉрЃА (рЃЏрЃљрЃњ. Windows 10, 11) рЃЊрЃљ рЃљрЃАрЃћрЃЋрЃћ рЃЏрЃўрЃАрЃў рЃњрЃљрЃарЃћрЃЏрЃЮрЃА рЃфрЃЋрЃџрЃљрЃЊрЃћрЃЉрЃў, рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃћрЃЉрЃў, рЃЉрЃўрЃЉрЃџрЃўрЃЮрЃЌрЃћрЃЎрЃћрЃЉрЃў рЃЊрЃљ рЃцрЃљрЃўрЃџрЃћрЃЉрЃўрЃА рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃљ.

**рЃЏрЃЌрЃљрЃЋрЃљрЃарЃў рЃЎрЃЮрЃЏрЃърЃЮрЃюрЃћрЃюрЃбрЃћрЃЉрЃў:**
- ­ЪЊѓ рЃцрЃљрЃўрЃџрЃБрЃарЃў рЃАрЃўрЃАрЃбрЃћрЃЏрЃљ (C:\, D:\ рЃЊрЃљ рЃљ.рЃе.)
- РџЎ№ИЈ рЃњрЃљрЃарЃћрЃЏрЃЮрЃА рЃфрЃЋрЃџрЃљрЃЊрЃћрЃЉрЃў (Environment Variables)
- ­ЪДЕ рЃЊрЃљрЃўрЃюрЃАрЃбрЃљрЃџрЃўрЃарЃћрЃЉрЃБрЃџрЃў рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃћрЃЉрЃў (.exe)
- ­ЪћЂ рЃърЃарЃЮрЃфрЃћрЃАрЃћрЃЉрЃў рЃЊрЃљ рЃАрЃћрЃарЃЋрЃўрЃАрЃћрЃЉрЃў (Services, Startup apps)

* Windows Environment рЃљрЃарЃўрЃА рЃЎрЃЮрЃЏрЃърЃўрЃБрЃбрЃћрЃарЃўрЃА рЃАрЃљрЃЏрЃБрЃерЃљрЃЮ рЃњрЃљрЃарЃћрЃЏрЃЮ, рЃАрЃљрЃЊрЃљрЃф рЃЏрЃБрЃерЃљрЃЮрЃЉрЃА рЃДрЃЋрЃћрЃџрЃљ рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃљ, рЃЏрЃљрЃЌ рЃерЃЮрЃарЃўрЃА Java, Python, QGIS рЃЊрЃљ GeoServer.

---

## ­ЪїЇ GeoServer Windows Environment-рЃерЃў

GeoServer рЃљрЃарЃўрЃА рЃдрЃўрЃљ рЃЎрЃЮрЃЊрЃўрЃА рЃАрЃћрЃарЃЋрЃћрЃарЃБрЃџрЃў рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃљ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃЏрЃБрЃерЃљрЃЮрЃЉрЃА Java рЃърЃџрЃљрЃбрЃцрЃЮрЃарЃЏрЃљрЃќрЃћ. GeoServer-рЃА рЃАрЃГрЃўрЃарЃЊрЃћрЃЉрЃљ:

- РюЁ рЃАрЃгрЃЮрЃарЃљрЃЊ рЃњрЃљрЃгрЃћрЃарЃўрЃџрЃў **Java рЃњрЃљрЃарЃћрЃЏрЃЮ** (JDK рЃљрЃю JRE)
- РюЁ рЃАрЃћрЃарЃЋрЃћрЃарЃБрЃџрЃў рЃАрЃўрЃЋрЃарЃфрЃћ (рЃАрЃљрЃЦрЃљрЃдрЃљрЃџрЃЊрЃћрЃћрЃЉрЃў, рЃЉрЃўрЃЉрЃџрЃўрЃЮрЃЌрЃћрЃЎрЃћрЃЉрЃў)
- РюЁ Windows-рЃўрЃА рЃерЃћрЃАрЃљрЃФрЃџрЃћрЃЉрЃџрЃЮрЃЉрЃљ **рЃАрЃЎрЃарЃўрЃърЃбрЃћрЃЉрЃўрЃА (BAT) рЃњрЃљрЃерЃЋрЃћрЃЉрЃўрЃАрЃљрЃЌрЃЋрЃўрЃА**

### GeoServer-рЃА Windows-рЃерЃў рЃљрЃЦрЃЋрЃА рЃЮрЃарЃў рЃњрЃљрЃерЃЋрЃћрЃЉрЃўрЃА рЃњрЃќрЃљ:
1. **ZIP рЃЋрЃћрЃарЃАрЃўрЃљ** РђЊ рЃњрЃљрЃерЃЋрЃћрЃЉрЃљ `startup.bat` рЃцрЃљрЃўрЃџрЃўрЃЌ  
2. **Windows Installer (EXE)** РђЊ рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃў рЃЊрЃљрЃДрЃћрЃюрЃћрЃЉрЃљ рЃарЃЮрЃњрЃЮрЃарЃф рЃАрЃћрЃарЃЋрЃўрЃАрЃў

---

## РџЎ№ИЈ Java рЃњрЃљрЃарЃћрЃЏрЃЮрЃА рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃљ (GeoServer-рЃўрЃАрЃЌрЃЋрЃўрЃА)

1. рЃњрЃљрЃЊрЃЏрЃЮрЃгрЃћрЃарЃћ Temurin OpenJDK:  
   ­ЪћЌ https://adoptium.net/en-GB/temurin/releases

2. рЃЊрЃљрЃљрЃДрЃћрЃюрЃћ JDK (рЃарЃћрЃЎрЃЮрЃЏрЃћрЃюрЃЊрЃћрЃЉрЃБрЃџрЃўрЃљ JDK 17)

3. рЃљрЃгрЃћрЃарЃћ рЃњрЃљрЃарЃћрЃЏрЃЮрЃА рЃфрЃЋрЃџрЃљрЃЊрЃў:

   - Variable Name: `JAVA_HOME`  
   - Variable Value: `C:\Program Files\Eclipse Adoptium\jdk-17.x.x`

4. рЃЊрЃљрЃљрЃЏрЃљрЃбрЃћ рЃћрЃА рЃњрЃќрЃљ рЃАрЃўрЃАрЃбрЃћрЃЏрЃБрЃа **Path** рЃфрЃЋрЃџрЃљрЃЊрЃерЃў:
   ```
   %JAVA_HOME%\bin
   ```

5. рЃњрЃљрЃЊрЃљрЃљрЃЏрЃЮрЃгрЃЏрЃћ `cmd`-рЃерЃў:

   ```bash
   java -version
   ```

## ­Ъџђ GeoServer-рЃўрЃА рЃњрЃљрЃерЃЋрЃћрЃЉрЃљ Windows-рЃќрЃћ

### рЃЋрЃљрЃарЃўрЃљрЃюрЃбрЃў 1: ZIP рЃЋрЃћрЃарЃАрЃўрЃљ

1. рЃњрЃљрЃЊрЃЏрЃЮрЃгрЃћрЃарЃћ GeoServer ZIP рЃљрЃарЃЦрЃўрЃЋрЃў:  
   ­ЪћЌ https://geoserver.org/download

2. рЃњрЃљрЃерЃљрЃџрЃћ рЃАрЃљрЃЦрЃљрЃдрЃљрЃџрЃЊрЃћрЃерЃў (рЃЏрЃљрЃњ: `C:\geoserver`)

3. рЃњрЃљрЃБрЃерЃЋрЃў `startup.bat` рЃцрЃљрЃўрЃџрЃў `bin` рЃАрЃљрЃЦрЃљрЃдрЃљрЃџрЃЊрЃўрЃЊрЃљрЃю

## ­ЪЊї рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў: рЃарЃљ рЃ«рЃЊрЃћрЃЉрЃљ Windows Environment-рЃерЃў

| рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃў | рЃљрЃдрЃгрЃћрЃарЃљ |
|----------|--------|
| `JAVA_HOME` | рЃњрЃљрЃарЃћрЃЏрЃЮрЃА рЃфрЃЋрЃџрЃљрЃЊрЃў, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃЏрЃўрЃБрЃЌрЃўрЃЌрЃћрЃЉрЃА Java-рЃА рЃЏрЃЊрЃћрЃЉрЃљрЃарЃћрЃЮрЃЉрЃљрЃА |
| `startup.bat` | Windows рЃАрЃЎрЃарЃўрЃърЃбрЃў GeoServer-рЃўрЃА рЃњрЃљрЃАрЃљрЃерЃЋрЃћрЃЉрЃљрЃЊ |
| `%PATH%` | рЃАрЃўрЃАрЃбрЃћрЃЏрЃБрЃарЃў рЃфрЃЋрЃџрЃљрЃЊрЃў, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃљрЃФрЃџрЃћрЃЋрЃА рЃгрЃЋрЃЊрЃЮрЃЏрЃљрЃА `java.exe`-рЃќрЃћ |
| `C:\geoserver\data_dir` | GeoServer-рЃўрЃА рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃА рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃљ |
| `localhost:8080/geoserver` | рЃЉрЃарЃљрЃБрЃќрЃћрЃарЃўрЃЊрЃљрЃю GeoServer UI-рЃќрЃћ рЃгрЃЋрЃЊрЃЮрЃЏрЃўрЃА рЃЏрЃўрЃАрЃљрЃЏрЃљрЃарЃЌрЃў |

---

## ­ЪДа рЃарЃљрЃбрЃЮрЃЏ рЃљрЃарЃўрЃА рЃћрЃА рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЋрЃљрЃюрЃў?

- Windows Environment-рЃА рЃАрЃгрЃЮрЃарЃў рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃљ рЃњрЃљрЃюрЃАрЃљрЃќрЃдрЃЋрЃарЃљрЃЋрЃА, рЃўрЃЏрЃБрЃерЃљрЃЋрЃћрЃЉрЃА рЃЌрЃБ рЃљрЃарЃљ GeoServer.
- Java рЃњрЃљрЃарЃћрЃЏрЃЮрЃА рЃњрЃљрЃарЃћрЃерЃћ GeoServer рЃЋрЃћрЃа рЃњрЃљрЃћрЃерЃЋрЃћрЃЉрЃљ.
- рЃАрЃбрЃБрЃЊрЃћрЃюрЃбрЃћрЃЉрЃЏрЃљ рЃБрЃюрЃЊрЃљ рЃўрЃфрЃЮрЃЊрЃюрЃћрЃю рЃарЃЮрЃњрЃЮрЃа рЃњрЃљрЃўрЃерЃЋрЃљрЃА рЃАрЃћрЃарЃЋрЃћрЃарЃБрЃџрЃў рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃћрЃЉрЃў рЃЊрЃљ рЃарЃЮрЃњрЃЮрЃа рЃўрЃЏрЃБрЃерЃљрЃЮрЃю рЃАрЃўрЃАрЃбрЃћрЃЏрЃБрЃа рЃфрЃЋрЃџрЃљрЃЊрЃћрЃЉрЃЌрЃљрЃю.
