
# QGIS-рЃерЃў XML рЃцрЃЮрЃарЃЏрЃљрЃбрЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ

**QGIS (Quantum GIS)** рЃЏрЃарЃљрЃЋрЃљрЃџ рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃљрЃерЃў рЃўрЃДрЃћрЃюрЃћрЃЉрЃА **XML рЃцрЃЮрЃарЃЏрЃљрЃбрЃА**, рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃљрЃЊ рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃўрЃАрЃљ рЃЊрЃљ рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃљрЃдрЃгрЃћрЃарЃўрЃАрЃЌрЃЋрЃўрЃА. рЃЦрЃЋрЃћрЃЏрЃЮрЃЌ рЃўрЃ«рЃўрЃџрЃљрЃЋрЃЌ рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃў рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃћрЃЉрЃўрЃА рЃЕрЃљрЃЏрЃЮрЃюрЃљрЃЌрЃЋрЃљрЃџрЃА.

---

## ­ЪћД 1. рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃцрЃљрЃўрЃџрЃћрЃЉрЃў (.qgs / .qgz)

- `.qgs` рЃљрЃарЃўрЃА XML рЃцрЃЮрЃарЃЏрЃљрЃбрЃўрЃА рЃцрЃљрЃўрЃџрЃў, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃўрЃюрЃљрЃ«рЃљрЃЋрЃА рЃЏрЃЌрЃћрЃџ рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃљрЃА:
  - рЃцрЃћрЃюрЃћрЃЉрЃўрЃА рЃгрЃДрЃљрЃарЃЮрЃћрЃЉрЃў
  - рЃАрЃбрЃўрЃџрЃћрЃЉрЃў (symbology)
  - рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃарЃћрЃЉрЃљ (CRS)
  - рЃЉрЃћрЃГрЃЊрЃЋрЃўрЃА рЃњрЃљрЃюрЃџрЃљрЃњрЃћрЃЉрЃљ
  - рЃћрЃбрЃўрЃЎрЃћрЃбрЃўрЃарЃћрЃЉрЃљ рЃЊрЃљ рЃАрЃ«рЃЋрЃљ

- `.qgz` рЃљрЃарЃўрЃА рЃерЃћрЃЎрЃБрЃЏрЃерЃБрЃџрЃў рЃЋрЃћрЃарЃАрЃўрЃљ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃерЃўрЃњрЃюрЃўрЃЌ рЃерЃћрЃўрЃфрЃљрЃЋрЃА `.qgs` XML рЃцрЃљрЃўрЃџрЃА.

**рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў:**
```xml
<qgis projectname="MyProject" version="3.40">
  <layer-tree-group name="Layers">
    <layer-tree-layer id="roads2020" name="Roads" ... />
  </layer-tree-group>
</qgis>
```

---

## ­ЪЌ║№ИЈ 2. SLD (Styled Layer Descriptor)

- **SLD** рЃљрЃарЃўрЃА OGC рЃАрЃбрЃљрЃюрЃЊрЃљрЃарЃбрЃў, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф XML рЃцрЃЮрЃарЃЏрЃљрЃбрЃерЃў рЃљрЃдрЃгрЃћрЃарЃА рЃцрЃћрЃюрЃћрЃЉрЃўрЃА рЃАрЃбрЃўрЃџрЃА.
- рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ рЃАрЃбрЃўрЃџрЃўрЃА рЃњрЃљрЃЊрЃљрЃАрЃљрЃфрЃћрЃЏрЃљрЃЊ рЃАрЃ«рЃЋрЃљ рЃърЃџрЃљрЃбрЃцрЃЮрЃарЃЏрЃћрЃЉрЃќрЃћ, рЃЏрЃљрЃњ.: GeoServer.

---

## ­Ъќ╝№ИЈ 3. QML рЃцрЃљрЃўрЃџрЃћрЃЉрЃў (QGIS Layer Style)

- **QML** рЃљрЃарЃўрЃА QGIS-рЃўрЃА рЃАрЃљрЃЎрЃБрЃЌрЃљрЃарЃў XML рЃАрЃбрЃўрЃџрЃўрЃА рЃцрЃЮрЃарЃЏрЃљрЃбрЃў.
- рЃўрЃюрЃљрЃ«рЃљрЃЋрЃА:
  - рЃАрЃўрЃЏрЃЉрЃЮрЃџрЃўрЃЎрЃљрЃА
  - рЃцрЃћрЃарЃўрЃА рЃЎрЃџрЃљрЃАрЃўрЃцрЃўрЃЎрЃљрЃфрЃўрЃљрЃА
  - рЃћрЃбрЃўрЃЎрЃћрЃбрЃўрЃарЃћрЃЉрЃўрЃА рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃћрЃЉрЃА
- рЃљрЃа рЃљрЃарЃўрЃА рЃўрЃњрЃўрЃЋрЃћ рЃарЃљрЃф SLD, рЃЏрЃљрЃњрЃарЃљрЃЏ рЃЮрЃарЃўрЃЋрЃћ XML рЃцрЃЮрЃарЃЏрЃљрЃбрЃўрЃљ.

---

## ­Ъќе№ИЈ 4. рЃЉрЃћрЃГрЃЊрЃЋрЃўрЃАсЃєсЃ│рЃърЃџрЃљрЃбрЃћрЃЉрЃў (.qpt)

- `.qpt` рЃцрЃљрЃўрЃџрЃћрЃЉрЃў рЃўрЃюрЃљрЃ«рЃљрЃЋрЃћрЃю рЃарЃБрЃЎрЃћрЃЉрЃўрЃА рЃЉрЃћрЃГрЃЊрЃЋрЃўрЃАсЃєсЃ│рЃърЃџрЃљрЃбрЃћрЃЉрЃА XML рЃцрЃЮрЃарЃЏрЃљрЃбрЃерЃў.
- рЃЏрЃЮрЃўрЃфрЃљрЃЋрЃА рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃћрЃЉрЃА: рЃарЃБрЃЎрЃљ, рЃџрЃћрЃњрЃћрЃюрЃЊрЃљ, рЃџрЃЮрЃњрЃЮ, рЃЏрЃљрЃАрЃерЃбрЃљрЃЉрЃў рЃЊрЃљ рЃљ.рЃе.

---

## ­ЪЊд 5. Processing Models (.model3)

- `.model3` рЃљрЃарЃўрЃА XML рЃцрЃЮрЃарЃЏрЃљрЃбрЃерЃў рЃерЃћрЃюрЃљрЃ«рЃБрЃџрЃў рЃЏрЃЮрЃЊрЃћрЃџрЃў, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ QGIS Processing Toolbox-рЃерЃў.
- рЃљрЃдрЃгрЃћрЃарЃА:
  - рЃўрЃюрЃАрЃбрЃарЃБрЃЏрЃћрЃюрЃбрЃћрЃЉрЃА
  - рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃћрЃЉрЃА
  - рЃБрЃарЃЌрЃўрЃћрЃарЃЌрЃЊрЃљрЃЎрЃљрЃЋрЃерЃўрЃарЃћрЃЉрЃљрЃА

**рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў:**
```xml
<model>
  <algorithm id="native:buffer">
    <parameter name="DISTANCE" value="10"/>
  </algorithm>
</model>
```

## ­Ъћї 6. Plugin Metadata

- рЃќрЃЮрЃњрЃ»рЃћрЃа QGIS рЃърЃџрЃљрЃњрЃўрЃюрЃћрЃЉрЃў рЃўрЃДрЃћрЃюрЃћрЃЉрЃћрЃю `metadata.xml` рЃцрЃљрЃўрЃџрЃА рЃЊрЃћрЃбрЃљрЃџрЃБрЃарЃў рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃўрЃАрЃЌрЃЋрЃўрЃА.
- рЃћрЃА XML рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ QGIS Plugin Repository-рЃерЃў рЃърЃџрЃљрЃњрЃўрЃюрЃўрЃА рЃарЃћрЃњрЃўрЃАрЃбрЃарЃљрЃфрЃўрЃўрЃАрЃљрЃА.

## ­ЪДЕ 7. UI рЃцрЃЮрЃарЃЏрЃўрЃАсЃєсЃ│рЃърЃџрЃљрЃбрЃћрЃЉрЃў (.ui)

- Qt Designer-рЃўрЃЌ рЃерЃћрЃЦрЃЏрЃюрЃўрЃџрЃў рЃцрЃЮрЃарЃЏрЃћрЃЉрЃў рЃўрЃюрЃљрЃ«рЃћрЃЉрЃљ `.ui` рЃњрЃљрЃцрЃљрЃарЃЌрЃЮрЃћрЃЉрЃўрЃА XML рЃцрЃљрЃўрЃџрЃћрЃЉрЃерЃў.
- рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ QGIS рЃљрЃбрЃарЃўрЃЉрЃБрЃбрЃћрЃЉрЃўрЃА рЃцрЃЮрЃарЃЏрЃўрЃА рЃЊрЃўрЃќрЃљрЃўрЃюрЃўрЃАрЃЌрЃЋрЃўрЃА.

| рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ                      | рЃњрЃљрЃцрЃљрЃарЃЌрЃЮрЃћрЃЉрЃљ       | рЃцрЃЮрЃарЃЏрЃљрЃбрЃў | рЃљрЃдрЃгрЃћрЃарЃљ |

рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃўрЃЌрЃў рЃЎрЃўрЃЌрЃ«рЃЋрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА рЃерЃћрЃњрЃўрЃФрЃџрЃўрЃљ рЃЏрЃЮрЃЏрЃгрЃћрЃарЃЮ рЃЎрЃЮрЃюрЃЎрЃарЃћрЃбрЃБрЃџрЃў рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃћрЃЉрЃўрЃА рЃљрЃю рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃА рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА.
