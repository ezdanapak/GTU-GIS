# ­ЪЊџ рЃарЃљ рЃљрЃарЃўрЃА GeoNetwork? РђЊ рЃЏрЃљрЃарЃбрЃўрЃЋрЃљрЃЊ рЃАрЃбрЃБрЃЊрЃћрЃюрЃбрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА

## ­ЪДЙ рЃЏрЃЮрЃЎрЃџрЃћрЃЊ: рЃарЃљ рЃљрЃарЃўрЃА GeoNetwork?

**GeoNetwork** рЃљрЃарЃўрЃА рЃдрЃўрЃљ рЃЎрЃЮрЃЊрЃўрЃА (open-source) рЃАрЃўрЃЋрЃарЃфрЃБрЃџрЃў рЃЏрЃћрЃбрЃљ-рЃЊрЃЮрЃюрЃћрЃћрЃЉрЃўрЃА рЃЏрЃљрЃарЃЌрЃЋрЃўрЃА рЃАрЃўрЃАрЃбрЃћрЃЏрЃљ.  
рЃўрЃА рЃћрЃ«рЃЏрЃљрЃарЃћрЃЉрЃљ рЃЮрЃарЃњрЃљрЃюрЃўрЃќрЃљрЃфрЃўрЃћрЃЉрЃА рЃЊрЃљ рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃћрЃЉрЃА **рЃЏрЃћрЃбрЃљ-рЃЊрЃЮрЃюрЃћрЃћрЃЉрЃўрЃА рЃФрЃўрЃћрЃЉрЃљрЃерЃў, рЃњрЃљрЃќрЃўрЃљрЃарЃћрЃЉрЃљрЃАрЃљ рЃЊрЃљ рЃЏрЃљрЃарЃЌрЃЋрЃљрЃерЃў**.

­Ъј» рЃЏрЃўрЃАрЃў рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃў рЃЏрЃўрЃќрЃљрЃюрЃўрЃљ:
> рЃњрЃћрЃЮрЃњрЃарЃљрЃцрЃўрЃБрЃџрЃў рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃА рЃБрЃЎрЃћрЃЌрЃћрЃАрЃў рЃЮрЃарЃњрЃљрЃюрЃўрЃќрЃћрЃЉрЃљ рЃЊрЃљ рЃЏрЃЮрЃФрЃўрЃћрЃЉрЃљ рЃўрЃюрЃбрЃћрЃарЃюрЃћрЃбрЃўрЃЌ.

---

## ­ЪћЇ рЃарЃљрЃА рЃюрЃўрЃерЃюрЃљрЃЋрЃА рЃЏрЃћрЃбрЃљ-рЃЊрЃЮрЃюрЃћ?

**рЃЏрЃћрЃбрЃљ-рЃЊрЃЮрЃюрЃћ** рЃюрЃўрЃерЃюрЃљрЃЋрЃА рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃўрЃА рЃљрЃдрЃгрЃћрЃарЃљрЃА.  
рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃљрЃЊ, рЃарЃБрЃЎрЃўрЃА рЃцрЃћрЃюрЃўрЃА рЃерЃћрЃАрЃљрЃ«рЃћрЃЉ рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃљ:
- рЃЋрЃўрЃю рЃерЃћрЃЦрЃЏрЃюрЃљ?
- рЃарЃЮрЃЊрЃўрЃА рЃерЃћрЃўрЃЦрЃЏрЃюрЃљ?
- рЃарЃљрЃА рЃгрЃљрЃарЃЏрЃЮрЃљрЃЊрЃњрЃћрЃюрЃА?
- рЃарЃљ рЃцрЃЮрЃарЃЏрЃљрЃбрЃерЃўрЃљ?
- рЃарЃЮрЃњрЃЮрЃа рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ?

­ЪЊї GeoNetwork рЃАрЃгрЃЮрЃарЃћрЃЊ рЃљрЃЏ рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃўрЃА рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃўрЃарЃћрЃЉрЃБрЃџрЃљрЃЊ рЃерЃћрЃюрЃљрЃ«рЃЋрЃљрЃќрЃћ рЃЊрЃљ рЃЏрЃЮрЃФрЃўрЃћрЃЉрЃљрЃќрЃћрЃљ рЃЮрЃарЃўрЃћрЃюрЃбрЃўрЃарЃћрЃЉрЃБрЃџрЃў.

---

## ­Ъїљ рЃарЃЮрЃњрЃЮрЃа рЃЏрЃБрЃерЃљрЃЮрЃЉрЃА GeoNetwork?

| рЃерЃћрЃАрЃљрЃФрЃџрЃћрЃЉрЃџрЃЮрЃЉрЃљ | рЃљрЃдрЃгрЃћрЃарЃљ |
|--------------|--------|
| ­Ъћј рЃФрЃћрЃЉрЃюрЃљ | рЃЏрЃЮрЃФрЃћрЃЉрЃюрЃћ рЃарЃБрЃЎрЃўрЃА рЃцрЃћрЃюрЃћрЃЉрЃў, рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃў, рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃћрЃЉрЃў |
| ­ЪЊІ рЃљрЃдрЃгрЃћрЃарЃљ | рЃюрЃљрЃ«рЃћ рЃцрЃћрЃюрЃўрЃА рЃљрЃЋрЃбрЃЮрЃарЃў, рЃџрЃўрЃфрЃћрЃюрЃќрЃўрЃљ, рЃбрЃћрЃњрЃћрЃЉрЃў, рЃљрЃЊрЃњрЃўрЃџрЃў |
| ­ЪЊЂ рЃЎрЃљрЃбрЃљрЃџрЃЮрЃњрЃћрЃЉрЃў | рЃЊрЃљрЃљрЃџрЃљрЃњрЃћ рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃў рЃЌрЃћрЃЏрЃљрЃбрЃБрЃарЃљрЃЊ |
| ­ЪЊц рЃћрЃЦрЃАрЃърЃЮрЃарЃбрЃў / рЃўрЃЏрЃърЃЮрЃарЃбрЃў | ISO, INSPIRE, Dublin Core рЃАрЃбрЃљрЃюрЃЊрЃљрЃарЃбрЃћрЃЉрЃўрЃЌ |
| ­ЪДЕ рЃўрЃюрЃбрЃћрЃњрЃарЃљрЃфрЃўрЃљ | рЃЏрЃБрЃерЃљрЃЮрЃЉрЃА GeoServer, GeoNode, CKAN рЃЊрЃљ рЃАрЃ«рЃЋрЃљ рЃАрЃўрЃАрЃбрЃћрЃЏрЃћрЃЉрЃЌрЃљрЃю |

---

## ­ЪДф рЃАрЃбрЃБрЃЊрЃћрЃюрЃбрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА: рЃарЃўрЃАрЃЌрЃЋрЃўрЃА рЃњрЃљрЃЏрЃЮрЃЋрЃўрЃДрЃћрЃюрЃЮрЃЌ GeoNetwork?

- ­ЪЊЂ рЃЏрЃЮрЃФрЃћрЃЉрЃюрЃЮрЃю рЃЎрЃЮрЃюрЃЎрЃарЃћрЃбрЃБрЃџрЃў рЃцрЃћрЃюрЃћрЃЉрЃў рЃљрЃю рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃў рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃАрЃЌрЃЋрЃўрЃА
- ­ЪЊЮ рЃљрЃдрЃгрЃћрЃарЃЮрЃю рЃЌрЃљрЃЋрЃўрЃљрЃюрЃЌрЃў рЃљрЃбрЃЋрЃўрЃарЃЌрЃБрЃџрЃў рЃцрЃћрЃюрЃћрЃЉрЃў рЃАрЃгрЃЮрЃарЃў рЃЏрЃћрЃбрЃљ-рЃЊрЃЮрЃюрЃћрЃћрЃЉрЃўрЃЌ
- ­ЪДа рЃўрЃАрЃгрЃљрЃЋрЃџрЃЮрЃю рЃАрЃљрЃћрЃарЃЌрЃљрЃерЃЮрЃарЃўрЃАрЃЮ рЃАрЃбрЃљрЃюрЃЊрЃљрЃарЃбрЃћрЃЉрЃў (ISO 19115)
- ­ЪїЇ рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃЮрЃю SDI (Spatial Data Infrastructure) рЃърЃарЃЮрЃћрЃЦрЃбрЃћрЃЉрЃерЃў

---

## ­ЪЉеРђЇ­Ъњ╗ рЃбрЃћрЃЦрЃюрЃЮрЃџрЃЮрЃњрЃўрЃБрЃарЃў рЃЏрЃ«рЃљрЃарЃћ

GeoNetwork рЃљрЃерЃћрЃюрЃћрЃЉрЃБрЃџрЃўрЃљ рЃерЃћрЃЏрЃЊрЃћрЃњрЃќрЃћ:
- Java рЃЊрЃљ Spring Framework
- XML рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃА рЃЊрЃљрЃЏрЃБрЃерЃљрЃЋрЃћрЃЉрЃљ
- PostgreSQL рЃЉрЃљрЃќрЃљ
- GeoServer-рЃЌрЃљрЃю рЃЎрЃљрЃЋрЃерЃўрЃарЃў (рЃЏрЃљрЃњ: рЃцрЃћрЃюрЃўрЃА рЃЏрЃћрЃбрЃљ-рЃЊрЃЮрЃюрЃћ)

---

## ­ЪЊџ рЃарЃўрЃЌ рЃњрЃљрЃюрЃАрЃ«рЃЋрЃљрЃЋрЃЊрЃћрЃЉрЃљ GeoNode-рЃАрЃњрЃљрЃю?

| рЃЏрЃљрЃ«рЃљрЃАрЃўрЃљрЃЌрЃћрЃЉрЃћрЃџрЃў | GeoNode | GeoNetwork |
|---------------|---------|------------|
| рЃцрЃћрЃюрЃўрЃА рЃАрЃћрЃарЃЋрЃўрЃюрЃњрЃў | РюЁ рЃЎрЃў | РЮї рЃљрЃарЃљ |
| рЃЏрЃћрЃбрЃљ-рЃЊрЃЮрЃюрЃћ | РюЁ рЃљрЃЦрЃЋрЃА, рЃЏрЃљрЃњрЃарЃљрЃЏ рЃБрЃцрЃарЃЮ рЃЏрЃљрЃарЃбрЃўрЃЋрЃў | РюЁ рЃЊрЃћрЃбрЃљрЃџрЃБрЃарЃў, рЃАрЃбрЃљрЃюрЃЊрЃљрЃарЃбрЃўрЃќрЃћрЃЉрЃБрЃџрЃў |
| рЃарЃБрЃЎрЃўрЃА рЃЋрЃўрЃќрЃБрЃљрЃџрЃўрЃќрЃљрЃфрЃўрЃљ | РюЁ рЃЎрЃў | ­ЪћИ рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃюрЃљрЃ«рЃЋрЃљ |
| рЃўрЃюрЃбрЃћрЃњрЃарЃљрЃфрЃўрЃљ SDI-рЃЌрЃљрЃю | рЃАрЃљрЃерЃБрЃљрЃџрЃЮ | РюЁ рЃЏрЃљрЃдрЃљрЃџрЃў |
| ISO рЃАрЃбрЃљрЃюрЃЊрЃљрЃарЃбрЃћрЃЉрЃў | рЃюрЃљрЃгрЃўрЃџрЃЮрЃЉрЃарЃўрЃЋ | РюЁ рЃАрЃарЃБрЃџрЃў рЃЏрЃ«рЃљрЃарЃЊрЃљрЃГрЃћрЃарЃљ |

РъА№ИЈ **GeoNode** рЃБрЃцрЃарЃЮ GIS рЃърЃџрЃљрЃбрЃцрЃЮрЃарЃЏрЃљрЃљ, рЃ«рЃЮрЃџрЃЮ **GeoNetwork** РђЊ рЃЏрЃћрЃбрЃљ-рЃЊрЃЮрЃюрЃћрЃћрЃЉрЃўрЃА рЃЎрЃљрЃбрЃљрЃџрЃЮрЃњрЃўрЃА рЃАрЃўрЃАрЃбрЃћрЃЏрЃљ.
